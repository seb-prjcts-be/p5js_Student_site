Een object bundelt meerdere gegevens onder één naam. Elke gegevenswaarde krijgt een sleutel (naam) en een waarde. Perfect om bij elkaar horende info samen te houden.

Kernpunten
  • Een object aanmaken met {} en sleutel/waarde-paren
  • Eigenschappen lezen met puntnotatie: object.eigenschap
  • Eigenschappen wijzigen of nieuwe toevoegen
  • Een object gebruiken om bv. positie, kleur en grootte samen op te slaan

Wat is een Object?

Met een array sla je meerdere waarden op, maar die hebben alleen een index (0, 1, 2...). Een object geeft elke waarde een naam:

let persoon = {
  naam: "frank",
  leeftijd: 11
};

Tussen de accolades staan sleutel: waarde-paren, gescheiden door een komma. De sleutel is de naam (bv. naam), de waarde is wat erin zit (bv. "frank").

Waarom Heet Het een "Object Literal"?

In het Nederlands: een letterlijk object. Je schrijft het object letterlijk uit tussen {}, zoals je het in je hoofd ziet. Een literal is dus een waarde die je rechtstreeks in de code noteert, zoals ze is:

  • 42 - number literal
  • "halo" - string literal
  • true - boolean literal
  • [1, 2, 3] - array literal
  • { naam: "Mila" } - object literal

Kortom: een object literal is gewoon een bundeltje gegevens onder één naam, rechtstreeks in de code genoteerd.

Eigenschappen Lezen

Met een punt haal je een eigenschap op:

console.log(persoon.naam);      // "frank"
console.log(persoon.leeftijd);  // 11

Eigenschappen Wijzigen of Toevoegen

Je kunt waarden aanpassen of extra eigenschappen bijmaken:

persoon.leeftijd = 12;               // Wijzig bestaande eigenschap
persoon.lievelingsKleur = "blauw";   // Voeg nieuwe eigenschap toe

console.log(persoon);
// { naam: "frank", leeftijd: 12, lievelingsKleur: "blauw" }

Objecten in p5.js

In p5 gebruik je objecten vaak om positie, grootte en kleur van iets samen te houden. Zo hoef je niet losse variabelen als balX, balY en balGrootte bij te houden:

let bal = {
  x: 300,
  y: 200,
  grootte: 60,
  kleur: [100, 150, 250]
};

function draw() {
  background(240);
  fill(bal.kleur);
  circle(bal.x, bal.y, bal.grootte);
}
