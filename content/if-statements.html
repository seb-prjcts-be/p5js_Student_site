Met if-statements maak je beslissingen in je code. Je code kan verschillende dingen doen afhankelijk van voorwaarden.

Basis If-Statement

Een if-statement controleert een voorwaarde:

if (voorwaarde) {
  // Code die uitgevoerd wordt als voorwaarde waar is
}

Voorbeeld

function draw() {
  if (mouseX > width/2) {
    fill(255, 0, 0);  // Rood als muis rechts
  } else {
    fill(0, 0, 255);  // Blauw als muis links
  }
  ellipse(200, 200, 100, 100);
}

Vergelijkingen

Je kunt verschillende vergelijkingen maken:

  • == - Gelijk aan
  • != - Niet gelijk aan
  • > - Groter dan
  • < - Kleiner dan
  • >= - Groter of gelijk
  • <= - Kleiner of gelijk

Voorbeelden

if (x > 100) { }
if (y < 200) { }
if (diameter == 50) { }
if (snelheid != 0) { }

Else en Else If

Je kunt meerdere voorwaarden controleren:

if (x < 100) {
  fill(255, 0, 0);  // Rood
} else if (x < 200) {
  fill(0, 255, 0);  // Groen
} else {
  fill(0, 0, 255);  // Blauw
}

Logische Operatoren

Combineer voorwaarden met && (en) en || (of):

if (x > 100 && x < 300) {
  // x is tussen 100 en 300
}

if (x < 50 || x > 350) {
  // x is kleiner dan 50 OF groter dan 350
}
