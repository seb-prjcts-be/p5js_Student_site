Variabelen slaan waarden op die je later kunt gebruiken. Ze zijn essentieel voor dynamische en interactieve sketches.

Variabelen Aanmaken

In JavaScript gebruik je let of const om variabelen aan te maken:

let x = 100;        // Kan veranderen
const y = 200;      // Blijft constant
let diameter = 50;  // Diameter van cirkel

Wanneer gebruik je wat?

  • let - Voor waarden die kunnen veranderen
  • const - Voor waarden die niet veranderen

Types Variabelen

JavaScript heeft verschillende datatypes:

Numbers (Getallen)

let x = 100;
let snelheid = 2.5;
let percentage = 0.75;

Strings (Tekst)

let naam = "p5.js";
let bericht = 'Hello World';

Booleans (True/False)

let isActief = true;
let isGepauzeerd = false;

Variabelen Gebruiken

Variabelen maken je code flexibel:

let x = 100;
let y = 200;
let diameter = 50;

function draw() {
  ellipse(x, y, diameter, diameter);
  
  // Verander waarden
  x += 1;  // Beweeg naar rechts
}

Berekeningen

Je kunt berekeningen doen met variabelen:

let breedte = 400;
let hoogte = 300;
let middenX = breedte / 2;      // 200
let middenY = hoogte / 2;       // 150
let oppervlakte = breedte * hoogte;  // 120000
