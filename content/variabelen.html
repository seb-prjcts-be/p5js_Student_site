Variabelen slaan waarden op die je later kunt gebruiken. Ze zijn essentieel voor dynamische en interactieve sketches.

Kernpunten
  • Variabelen declareren met let (veranderlijk) en const (vast)
  • De basistypes herkennen: integer, float, string, boolean
  • Het verschil tussen globale en lokale scope
  • Ingebouwde systeemvariabelen zoals width, height, windowWidth, windowHeight
  • Waarden onthouden tussen frames door variabelen buiten setup() en draw() te zetten

Variabelen Aanmaken

In JavaScript gebruik je let of const om variabelen aan te maken:

let x = 100;        // Kan veranderen
const y = 200;      // Blijft constant
let diameter = 50;  // Diameter van cirkel

Wanneer gebruik je wat?

  • let - Voor waarden die kunnen veranderen
  • const - Voor waarden die niet veranderen

Types variabelen

JavaScript kent verschillende soorten waarden. In p5.js steekt het meestal niet zo nauw, maar het is goed om de types te kennen:

Integer (geheel getal)

let leeftijd = 25;
let x = -5;

Float (kommagetal)

Gebruik een punt voor decimalen, geen komma:

let snelheid = 2.5;
let gewicht = 68.5;

String (tekst)

Tekst staat altijd tussen aanhalingstekens - enkel of dubbel:

let naam = "Jan";
let geboortedatum = "10/04/1984";
let bericht = 'Hello World';

String samenvoegen (concateneren)

Tekst en variabelen kun je samenvoegen met een +-teken:

let naam = "Els";
let x = 100;
let zin = naam + " heeft " + x + " appels.";
console.log(zin);
// → Els heeft 100 appels.
Let op

Het totale aantal aanhalingstekens moet altijd even zijn.

Boolean (waar/onwaar)

Een boolean heeft maar twee mogelijke waarden: true of false. Met een uitroepteken ! kun je de waarde omdraaien:

let isActief = true;
let isGepauzeerd = false;

isActief = !isActief;  // Nu false

Numbers (getallen)

let x = 100;
let snelheid = 2.5;
let percentage = 0.75;

Variabelen Gebruiken

Variabelen maken je code flexibel:

let x = 100;
let y = 200;
let diameter = 50;

function draw() {
  ellipse(x, y, diameter, diameter);
  
  // Verander waarden
  x += 1;  // Beweeg naar rechts
}

Waar leeft een variabele?

Niet elke variabele is overal beschikbaar. Dat hangt af van waar je hem declareert:

  • Buiten alle functies → globaal: beschikbaar in heel je sketch
  • Binnen een functie → lokaal: bestaat alleen zolang die functie loopt
let globaleX = 0;  // Beschikbaar in setup(), draw(), en overal

function draw() {
  let lokaalY = 100;  // Bestaat alleen binnen draw()
  globaleX += 1;      // Werkt - globaleX is van buiten

  // lokaalY is hier nog beschikbaar...
}
// ...maar hier niet meer

// globaleX bestaat nog steeds

In p5.js declareer je variabelen die je in meerdere functies nodig hebt - zoals posities, snelheden of tellers - altijd buiten setup() en draw(). Dan onthouden ze hun waarde van frame tot frame.

let x = 0;          // Onthouden tussen frames

function setup() {
  createCanvas(600, 400);
}

function draw() {
  background(245);
  circle(x, 200, 40);
  x += 2;           // x groeit elk frame
}

Ingebouwde systeemvariabelen

p5.js beheert een aantal variabelen automatisch voor jou. Je hoeft ze niet zelf aan te maken - ze zijn altijd beschikbaar:

width en height

De breedte en hoogte van het canvas dat je in createCanvas() hebt ingesteld:

function draw() {
  let middenX = width / 2;   // Horizontaal midden van het canvas
  let middenY = height / 2;  // Verticaal midden
  circle(middenX, middenY, 100);
}

windowWidth en windowHeight

De breedte en hoogte van het browservenster (niet het canvas). Handig om een canvas te maken dat het hele venster vult:

function setup() {
  createCanvas(windowWidth, windowHeight);
}

// Aanpassen als het venster vergroot of verkleind wordt:
function windowResized() {
  resizeCanvas(windowWidth, windowHeight);
}

Berekeningen

Met variabelen kun je rekenen: optellen, aftrekken, vermenigvuldigen, delen. De vier basisbewerkingen, de modulo %, en handige functies zoals map() en dist() worden uitgebreid behandeld op de pagina Berekeningen.

let middenX = width / 2;
let middenY = height / 2;
let rood = map(mouseX, 0, width, 0, 255);  // schaal muis naar kleur
