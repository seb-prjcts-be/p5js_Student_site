Game of Life is een cellulair automaat van John Conway. Er zijn geen spelers en geen doel: alleen cellen die leven of sterven volgens vier regels.

Kernpunten
  • Werken met rasters en burenrelaties in een 2D grid
  • Toestanden berekenen over opeenvolgende generaties
  • Zien hoe vier lokale regels globale patronen laten ontstaan
  • Simulaties opbouwen met twee grids: huidig en volgend
  • Patronen herkennen zoals still lifes, oscillatoren en gliders

Waarom dit concept zo beroemd is

Game of Life laat zien hoe complex gedrag kan ontstaan uit bijna niets. Met vier simpele regels krijg je patronen die pulseren, bewegen, botsen of verdwijnen.

Denkvraag: wanneer voelt een systeem "levend", ook al leeft er niets echt?

De regels

Elke cel is levend (1) of dood (0). Elke generatie verandert op basis van zijn 8 buren:

  • Levende cel met 2 of 3 buren - overleeft
  • Levende cel met minder dan 2 buren - sterft door isolatie
  • Levende cel met meer dan 3 buren - sterft door overbevolking
  • Dode cel met exact 3 buren - wordt levend

Die vier regels zijn genoeg voor still lifes, oscillatoren en gliders.

Structuur in p5.js

Je werkt met twee grids: het huidige en het volgende. Zo voorkom je dat een verandering midden in de berekening andere cellen meteen beinvloedt.

for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let n = telBuren(grid, i, j);
    let cel = grid[i][j];
    if (cel === 1 && (n === 2 || n === 3)) next[i][j] = 1;
    else if (cel === 0 && n === 3)         next[i][j] = 1;
    else                                    next[i][j] = 0;
  }
}

[grid, next] = [next, grid];

Buren tellen met wrapping

Met wrapping sluit de rechterkant aan op de linkerkant en de bovenkant op de onderkant:

function telBuren(g, x, y) {
  let sum = 0;
  for (let i = -1; i <= 1; i++) {
    for (let j = -1; j <= 1; j++) {
      if (i === 0 && j === 0) continue;
      sum += g[(x + i + cols) % cols][(y + j + rows) % rows];
    }
  }
  return sum;
}

Patronen herkennen

In Game of Life krijgen vormen bijna een eigen persoonlijkheid. Sommige blijven staan, sommige knipperen, sommige schuiven door het raster.

  • still life - verandert niet meer
  • oscillator - herhaalt zichzelf na enkele stappen
  • glider - beweegt schuin door het grid
