Game of Life is een cellulair automaat bedacht door wiskundige John Conway in 1970. Er zijn geen spelers — het systeem evolueert zichzelf op basis van vier simpele regels.

De regels

Elk cel is levend (1) of dood (0). Elke generatie veranderen cellen op basis van hun 8 buren:

  • Levende cel met 2 of 3 buren — overleeft
  • Levende cel met <2 buren — sterft (isolatie)
  • Levende cel met >3 buren — sterft (overbevolking)
  • Dode cel met exact 3 buren — wordt levend (geboorte)

Die vier regels zijn genoeg voor eindeloos complexe patronen: stabiele vormen, oscillatoren, en zelfs "gliders" die over het raster bewegen.

Structuur in p5.js

Je werkt met twee grids: het huidige en het volgende. Je berekent de volgende generatie altijd op basis van het huidige grid, anders beïnvloeden cellen elkaar tijdens de berekening.

// Bereken volgende generatie
for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let n = telBuren(grid, i, j);
    let cel = grid[i][j];
    if (cel === 1 && (n === 2 || n === 3)) next[i][j] = 1;
    else if (cel === 0 && n === 3)         next[i][j] = 1;
    else                                    next[i][j] = 0;
  }
}
// Wissel grids
[grid, next] = [next, grid];

Buren tellen met wrapping

Om botsing met de randen te vermijden, "wrappen" we de randen — de rechterkant sluit aan op de linkerkant:

function telBuren(g, x, y) {
  let sum = 0;
  for (let i = -1; i <= 1; i++)
    for (let j = -1; j <= 1; j++) {
      if (i === 0 && j === 0) continue;
      sum += g[(x + i + cols) % cols][(y + j + rows) % rows];
    }
  return sum;
}
