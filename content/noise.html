Noise is een speciale wiskundige functie die vloeiende, "organische" willekeur genereert. Waar random() elk frame een compleet willekeurige waarde geeft, geeft noise() waarden die rustig in elkaar overlopen - ideaal voor vloeiende beweging, texturen en landschappen.

Weetje

De noise() die je hier gebruikt heet voluit Perlin noise, bedacht door Ken Perlin in 1983 terwijl hij aan de speciale effecten van de film Tron (1982) werkte. Hij zocht een manier om levendige texturen te maken zonder dat ze er digitaal of geblokt uitzagen. Zijn uitvinding werd een standaardtechniek in games, films en generatieve kunst. In 1997 kreeg hij er een Academy Award voor Technical Achievement voor.

Kernpunten
  • Verschil tussen random() (schokkerig) en noise() (vloeiend)
  • noise(t) geeft altijd een waarde tussen 0 en 1 - stap traag door t voor vloeiende beweging
  • Waarden omzetten naar een bereik met map()
  • 2D noise noise(x, y) voor texturen, wolken en landschappen
  • noiseSeed() voor herhaalbare patronen en noiseDetail() voor meer octaven

noise() vs random()

Het verschil is meteen zichtbaar:

// random(): springt elke frame naar een willekeurige waarde
let x = random(0, width);   // Schokkerig, onrustig

// noise(): vloeiende overgang tussen waarden
let offset = 0;

function draw() {
  let x = noise(offset) * width;  // noise() geeft waarde tussen 0 en 1
  offset += 0.01;                 // Kleine stap = vloeiende beweging
}

noise(t) geeft altijd een waarde tussen 0 en 1. Door t langzaam te verhogen, beweeg je door de noise-ruimte.

Basisgebruik

let t = 0;

function draw() {
  background(245);

  let x = noise(t) * width;
  let y = noise(t + 100) * height;  // Ander startpunt = andere beweging

  circle(x, y, 40);

  t += 0.01;  // Snelheid: kleiner = trager en vloeiender
}

Door t + 100 voor de y-coördinaat te gebruiken, zorgen we dat x en y onafhankelijk van elkaar bewegen.

noise() Mappen naar een Bereik

noise() geeft altijd een waarde tussen 0 en 1. Gebruik map() om dat om te zetten naar een gewenst bereik:

let t = 0;

function draw() {
  let diameter = map(noise(t), 0, 1, 20, 200);
  // of: let diameter = noise(t) * 200;

  background(245);
  circle(width/2, height/2, diameter);
  t += 0.02;
}

2D Noise: Texturen

noise() werkt ook in twee dimensies: noise(x, y). Zo maak je naadloze texturen en "wolken":

function setup() {
  createCanvas(400, 400);
  noLoop();  // Slechts één keer tekenen
}

function draw() {
  loadPixels();
  let schaal = 0.005;  // Kleiner = grotere "vlekken"

  for (let x = 0; x < width; x++) {
    for (let y = 0; y < height; y++) {
      let n = noise(x * schaal, y * schaal);
      let helderheid = n * 255;
      let index = (x + y * width) * 4;
      pixels[index]     = helderheid;  // R
      pixels[index + 1] = helderheid;  // G
      pixels[index + 2] = helderheid;  // B
      pixels[index + 3] = 255;         // Alpha
    }
  }
  updatePixels();
}

noiseSeed() en noiseDetail()

Je kunt het noise-patroon beïnvloeden:

// noiseSeed(): zelfde seed = altijd hetzelfde patroon
noiseSeed(42);

// noiseDetail(): hoeveel "lagen" van detail (octaven)
// Meer octaven = meer detail maar grilliger
noiseDetail(8, 0.5);  // 8 octaven, amplitude halveert per octaaf

Flow field: noise als richtingen

De bekendste toepassing van noise in generatieve kunst is een flow field. Het idee is simpel: op elke plek (x, y) van het canvas vragen we een noise-waarde en gebruiken die als hoek. Deeltjes volgen die hoeken, alsof ze drijven op een onzichtbare stroom.

let deeltjes = [];
let schaal = 0.005;  // Kleiner = grotere wervels

function setup() {
  createCanvas(600, 400);
  background(10);
  for (let i = 0; i < 200; i++) {
    deeltjes.push({ x: random(width), y: random(height) });
  }
}

function draw() {
  background(10, 15);  // Spoor vervagen

  stroke(255, 150);
  strokeWeight(1);

  for (let d of deeltjes) {
    // Noise op (x, y) → hoek tussen 0 en 4π
    let hoek = noise(d.x * schaal, d.y * schaal) * TWO_PI * 2;

    let nx = d.x + cos(hoek) * 1.5;
    let ny = d.y + sin(hoek) * 1.5;

    line(d.x, d.y, nx, ny);

    d.x = nx;
    d.y = ny;

    // Nieuwe startpositie als een deeltje het canvas verlaat
    if (d.x < 0 || d.x > width || d.y < 0 || d.y > height) {
      d.x = random(width);
      d.y = random(height);
    }
  }
}

Noise Landschap

Noise is perfect voor terrein en golven:

let t = 0;

function draw() {
  background(30, 100, 180);  // Lucht

  fill(50, 180, 80);  // Groen terrein
  noStroke();
  beginShape();
  vertex(0, height);

  for (let x = 0; x <= width; x += 5) {
    let y = noise(x * 0.003 + t) * height * 0.7 + height * 0.2;
    vertex(x, y);
  }

  vertex(width, height);
  endShape(CLOSE);

  t += 0.003;  // Laat terrein langzaam schuiven
}

Voorbeeld: Flow field

Deeltjes volgen de hoeken die op elke plek uit noise(x, y) komen. Samen vormen hun sporen stromende curven.
