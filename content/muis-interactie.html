Muis interactie maakt je sketches responsief. p5.js heeft ingebouwde variabelen en functies voor muis input.

Muis Positie

p5.js heeft ingebouwde variabelen voor muispositie:

  • mouseX - X-positie van de muis
  • mouseY - Y-positie van de muis
  • pmouseX - Vorige X-positie
  • pmouseY - Vorige Y-positie

Voorbeeld

function draw() {
  background(245);
  ellipse(mouseX, mouseY, 50, 50);
}

Muis Events

Je kunt reageren op muisklikken en bewegingen:

mousePressed()

function mousePressed() {
  // Wordt uitgevoerd bij muisklik
  fill(random(255), random(255), random(255));
}

mouseDragged()

function mouseDragged() {
  // Wordt uitgevoerd tijdens slepen
  ellipse(mouseX, mouseY, 20, 20);
}

mouseReleased()

function mouseReleased() {
  // Wordt uitgevoerd bij loslaten
  background(245);
}

Pointer input: muis én touch

Pointer variabelen werken op zowel muis als touchscreen. Handig als een sketch op laptop én tablet moet werken.

Pointer variabelen

  • pointerX / pointerY - huidige positie
  • ppointerX / ppointerY - vorige positie
  • pointerIsDown - true zolang er gedrukt wordt
function draw() {
  background(245);
  fill(255, 120, 0);
  circle(pointerX, pointerY, 60);

  if (pointerIsDown) {
    fill(0, 180, 120);
    circle(pointerX, pointerY, 100);
  }
}

Pointer events

function pointerPressed() {
  fill(random(255), random(255), random(255));
  circle(pointerX, pointerY, 40);
}

function pointerMoved() {
  stroke(0, 140, 255);
  strokeWeight(3);
  line(ppointerX, ppointerY, pointerX, pointerY);
}

Je zult mousePressed() en touchStarted() nog veel tegenkomen in bestaand lesmateriaal. Voor nieuwe sketches die op meerdere toestellen moeten werken, zijn pointer events de eerste keuze.

Trails Maken

Met pmouseX en pmouseY kun je trails maken:

function draw() {
  // Teken lijn van vorige naar huidige muispositie
  line(pmouseX, pmouseY, mouseX, mouseY);
}
