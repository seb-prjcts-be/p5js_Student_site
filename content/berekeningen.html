p5.js is een rekenmachine die tekent. Bijna alles wat je ziet - posities, maten, kleuren, beweging - is het resultaat van berekeningen. Hoe je die berekeningen schrijft, bepaalt wat er op het scherm verschijnt.

Kernpunten
  • De vier basisoperatoren en voorrang met haakjes
  • Samengestelde operatoren (+=, -=, *=, /=) en x++ / x--
  • De modulo-operator % voor omslaan, ritme en raster
  • Willekeurige waarden met random(), inclusief integers via floor(random())
  • Kernfuncties: constrain(), map(), dist(), min(), max(), floor(), round()

De console als rekenmachine

Voordat je berekeningen in je sketch stopt, kun je ze uittesten met console.log(). Het is de snelste manier om te checken of een expressie doet wat je verwacht:

console.log(10 / 3);          // 3.333...
console.log(floor(10 / 3));   // 3
console.log(width / 2);       // Helft van het canvas

Open de console in je browser (F12) of in de p5.js-editor om het resultaat te zien. Gebruik het als verkenningsinstrument: probeer expressies uit voordat je ze in je code zet.

De vier basisoperatoren

let a = 10 + 3;   // 13  optellen
let b = 10 - 3;   // 7   aftrekken
let c = 10 * 3;   // 30  vermenigvuldigen
let d = 10 / 4;   // 2.5 delen

Volgorde van bewerkingen werkt zoals in wiskunde: vermenigvuldigen en delen vóór optellen en aftrekken. Gebruik haakjes om de volgorde te sturen:

let x = 2 + 3 * 4;    // 14  (niet 20)
let y = (2 + 3) * 4;  // 20

Samengestelde operatoren

Verkorte schrijfwijzen voor veelgebruikte bewerkingen:

let x = 10;

x += 5;   // x = x + 5  → 15
x -= 3;   // x = x - 3  → 12
x *= 2;   // x = x * 2  → 24
x /= 4;   // x = x / 4  → 6

x++;      // x = x + 1  → 7  (één ophogen)
x--;      // x = x - 1  → 6  (één verlagen)

In p5.js zie je x += snelheid en hoek += 0.02 overal terug - het is de standaardmanier om iets te laten bewegen of draaien.

De modulo-operator: %

% geeft de rest na deling terug. Denk aan klokrekenen: na 59 seconden kom je terug bij 0.

7 % 3   // → 1  (7 ÷ 3 = 2, rest 1)
10 % 5  // → 0  (10 ÷ 5 = 2, rest 0)
4 % 2   // → 0  (even: rest is altijd 0)
5 % 2   // → 1  (oneven: rest is altijd 1)

Gebruik % om een waarde te laten omslaan zodra die een grens bereikt:

let x = 0;

function draw() {
  x += 3;
  circle(x % width, height / 2, 40);  // slaat vanzelf om aan de rand
}

Of om iets te doen op een vast ritme:

function draw() {
  // Elke 60 frames (≈ elke seconde bij 60fps)
  if (frameCount % 60 === 0) {
    background(random(255));
  }

  // Wisselend per frame: even of oneven
  if (frameCount % 2 === 0) {
    fill(255, 0, 0);
  } else {
    fill(0, 0, 255);
  }
}

En in loops om rijen en kolommen te berekenen:

for (let i = 0; i < 12; i++) {
  let kolom = i % 4;       // 0, 1, 2, 3, 0, 1, 2, 3, ...
  let rij   = floor(i / 4); // 0, 0, 0, 0, 1, 1, 1, 1, ...
  circle(kolom * 100 + 50, rij * 100 + 50, 60);
}

random() - willekeurige getallen

random() geeft elke keer een ander kommagetal terug. Daardoor krijg je steeds een ander resultaat - essentieel voor generatieve kunst.

random(100)       // getal van 0 tot 99.99...  (nooit precies 100!)
random(50, 101)   // getal van 50 tot 100.99...

fill(random(255), random(255), random(255));  // Willekeurige kleur

Geheel getal nodig?

Wil je een willekeurig geheel getal, gebruik dan floor() om het naar beneden af te ronden:

// Willekeurig getal van 0 t/m 100 (inclusief 100)
let n = floor(random(101));

// Muntje opgooien: 0 of 1
let kant = floor(random(2));  // 0 = kop, 1 = munt
Let op

random(100) kan nooit precies 100 teruggeven, dus schrijf floor(random(101)) als je ook 100 wilt kunnen krijgen.

Rekenfuncties in p5.js

p5.js heeft een reeks ingebouwde functies die je veel schrijfwerk besparen.

Afronden

round(3.7)   // → 4   (dichtstbijzijnde geheel)
floor(3.9)   // → 3   (altijd naar beneden)
ceil(3.1)    // → 4   (altijd naar boven)
abs(-5)      // → 5   (absolute waarde, altijd positief)

Minimum en maximum

min(3, 7, 1)    // → 1  (kleinste waarde)
max(3, 7, 1)    // → 7  (grootste waarde)

constrain() - houd een waarde binnen grenzen

Handig om te voorkomen dat iets buiten het scherm gaat:

let x = constrain(mouseX, 50, width - 50);
// x is nooit kleiner dan 50, nooit groter dan width - 50

map() - vertaal het ene bereik naar het andere

map() is een van de meest gebruikte functies in generatieve kunst. Het kernidee: je hebt een waarde in bereik A en wilt die omzetten naar bereik B.

// map(waarde, van_min, van_max, naar_min, naar_max)

// mouseX (0 tot width) → kleurwaarde (0 tot 255)
let rood = map(mouseX, 0, width, 0, 255);
fill(rood, 0, 150);

// frameCount naar een hoek (0 tot TWO_PI)
let hoek = map(frameCount % 120, 0, 120, 0, TWO_PI);

// Je kunt ook omdraaien: groot wordt klein
let grootte = map(mouseY, 0, height, 80, 10);  // Bovenaan groot, onderaan klein
Tip

Bijna elk interactief of generatief effect begint met map(): muispositie naar kleur, afstand naar grootte, tijd naar rotatie. Als je één functie goed leert kennen, laat het deze zijn.

dist() - afstand tussen twee punten

// Afstand van de muis tot het midden
let d = dist(mouseX, mouseY, width / 2, height / 2);

// Groter als de muis verder weg is
let grootte = map(d, 0, 300, 10, 80);
circle(width / 2, height / 2, grootte);

sqrt() en sq()

sqrt(16)   // → 4   (vierkantswortel)
sq(4)      // → 16  (kwadraat, zelfde als 4 * 4)
