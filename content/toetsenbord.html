Met toetsenbord input kun je je sketches besturen. p5.js heeft functies om toetsaanslagen te detecteren.

keyPressed()

De keyPressed() functie wordt aangeroepen wanneer een toets wordt ingedrukt:

let kleur = 0;

function keyPressed() {
  if (key === 'r' || key === 'R') {
    kleur = color(255, 0, 0);  // Rood
  } else if (key === 'g' || key === 'G') {
    kleur = color(0, 255, 0);  // Groen
  }
}

function draw() {
  fill(kleur);
  ellipse(200, 200, 100, 100);
}

Toets Variabelen

p5.js heeft variabelen voor toetsen:

  • key - De ingedrukte toets (als karakter)
  • keyCode - Voor speciale toetsen

keyIsPressed

Je kunt controleren of een toets ingedrukt is:

function draw() {
  if (keyIsPressed) {
    if (key === 'w' || key === 'W') {
      y -= 2;  // Beweeg omhoog
    }
  }
}
