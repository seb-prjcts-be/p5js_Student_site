Turtle geometry is een manier van tekenen waarbij je een denkbeeldige schildpad bestuurt: zet hem vooruit, draai hem, en hij trekt een lijn. Eenvoudige commando's leiden tot verrassend complexe figuren.

Oorsprong: Logo en Seymour Papert

Turtle graphics werden in 1967 uitgevonden als onderdeel van de programmeertaal Logo, door Seymour Papert aan het MIT. Het idee was simpel: een kind kan zichzelf identificeren met de schildpad en zo "lichamelijk" wiskunde leren.

De schildpad heeft drie eigenschappen:

  • Positie — x en y op het canvas
  • Richting — de hoek waarnaar hij kijkt (heading)
  • Pen — neer (tekent) of op (beweegt zonder lijn)

De basiscommando's

forward(n)    // Ga n pixels vooruit (en teken een lijn)
right(hoek)   // Draai n graden naar rechts
left(hoek)    // Draai n graden naar links
penUp()       // Pen omhoog: beweeg zonder te tekenen
penDown()     // Pen omlaag: begin weer te tekenen

Implementatie in p5.js

De turtle houdt zijn positie en richting bij. forward() berekent het eindpunt met sinus en cosinus:

let tx, ty, heading, penNeer;

function forward(n) {
  let nx = tx + n * cos(radians(heading));
  let ny = ty + n * sin(radians(heading));
  if (penNeer) line(tx, ty, nx, ny);
  tx = nx;
  ty = ny;
}

function right(hoek) { heading += hoek; }
function left(hoek)  { heading -= hoek; }

Vormen tekenen

Een vierkant: vier keer vooruit, dan 90° draaien.

for (let i = 0; i < 4; i++) {
  forward(100);
  right(90);
}

Een ster met vijf punten: de buitenhoek is 144°.

for (let i = 0; i < 5; i++) {
  forward(120);
  right(144);
}

Een regelmatige n-hoek: de buitenhoek is altijd 360°/n.

let n = 6; // Zeshoek
for (let i = 0; i < n; i++) {
  forward(80);
  right(360 / n);
}

Spiralen

Vergroot de staplengte elke iteratie een beetje, dan krijg je een spiraal:

for (let i = 0; i < 200; i++) {
  forward(i * 0.5);  // Steeds groter
  right(91);         // Net niet 90° → spiraal
}

Het kleine verschil tussen de hoek en een "nette" veelvoud van 360° bepaalt hoe snel de spiraal uitdraait.

Recursieve patronen

Met een recursieve functie kun je de turtle zichzelf laten herhalen op kleinere schaal — zo ontstaan fractals:

function boom(lengte) {
  if (lengte < 5) return;
  forward(lengte);
  right(30);
  boom(lengte * 0.7);
  left(60);
  boom(lengte * 0.7);
  right(30);
  forward(-lengte); // Terug
}
boom(80);
