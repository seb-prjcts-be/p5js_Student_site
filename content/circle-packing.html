Circle packing is een algoritme dat probeert zo veel mogelijk niet-overlappende cirkels in een ruimte te plaatsen. Elke cirkel groeit vanuit een willekeurig punt totdat hij een andere cirkel of de rand raakt. Het resultaat is een dicht, organisch patroon dat tegelijk wiskundig precies en visueel aantrekkelijk is.

Hoe werkt het algoritme?

Het principe is eenvoudig:

  1. Kies een willekeurig punt in het canvas.
  2. Controleer of dat punt ver genoeg ligt van alle bestaande cirkels.
  3. Zo ja: voeg een nieuwe kleine cirkel toe op dat punt.
  4. Elke frame laat je alle groeiende cirkels iets groter worden.
  5. Een cirkel stopt met groeien zodra hij een andere cirkel of de rand raakt.

Je herhaalt stap 1–3 elke frame een paar keer. Na verloop van tijd is de ruimte zo vol dat er geen nieuwe cirkels meer passen — dan is het packing voltooid.

Overlapping detecteren met dist()

De kern van het algoritme is controleren of twee cirkels overlappen. Twee cirkels overlappen als de afstand tussen hun middelpunten kleiner is dan de som van hun stralen:

// Overlappen cirkel A en cirkel B?
let afstand = dist(a.x, a.y, b.x, b.y);
if (afstand < a.r + b.r) {
  // Ze overlappen
}

Om een veilige marge te houden voeg je een kleine buffer toe:

if (afstand < a.r + b.r + 2) {
  // Te dicht — niet plaatsen
}

Een cirkel opslaan als object

Elke cirkel heeft een positie, een straal en een vlag die bijhoudt of hij nog groeit:

let cirkels = [];

function voegToeCirkel(x, y) {
  cirkels.push({ x: x, y: y, r: 2, groeit: true });
}

function tekenEnGroei() {
  for (let c of cirkels) {
    circle(c.x, c.y, c.r * 2);

    if (c.groeit) {
      c.r += 0.5;
      if (raaktRand(c) || raaktAnder(c)) {
        c.r -= 0.5; // stap terug
        c.groeit = false;
      }
    }
  }
}

Rand en buurcirkels controleren

function raaktRand(c) {
  return c.x - c.r <= 0 || c.x + c.r >= width ||
         c.y - c.r <= 0 || c.y + c.r >= height;
}

function raaktAnder(c) {
  for (let ander of cirkels) {
    if (ander === c) continue;
    if (dist(c.x, c.y, ander.x, ander.y) < c.r + ander.r + 1) {
      return true;
    }
  }
  return false;
}

De check ander === c is belangrijk: een cirkel mag zichzelf niet als buur beschouwen.
