p5.js heeft ingebouwde functies voor basisvormen. Met deze functies kun je snel beginnen met tekenen.

Basiselementen: punt en lijn

De eenvoudigste vormen in p5.js zijn een punt en een lijn.

point() — Punt

Een punt is de eenvoudigste vorm, enkel gedefinieerd door zijn x- en y-coördinaten.

point(x, y);
point(100, 100);

line() — Lijn

Een lijn heeft twee eindpunten, elk met een x- en y-coördinaat.

line(x1, y1, x2, y2);
line(0, 100, 400, 100);

Ronde vormen: circle() en ellipse()

circle() — Cirkel

Een cirkel gebruik je wanneer breedte en hoogte gelijk zijn. Je geeft het middelpunt en de diameter op:

circle(x, y, diameter);
circle(200, 200, 150);  // Cirkel met diameter 150

ellipse() — Ovaal

Een ellips heeft een aparte breedte en hoogte. Als ze gelijk zijn krijg je ook een cirkel:

ellipse(x, y, breedte, hoogte);
ellipse(200, 200, 100, 100);  // Cirkel via ellipse
ellipse(200, 200, 150, 100);  // Ovaal

Rechthoekige vormen: square() en rect()

square() — Vierkant

Een vierkant gebruik je wanneer alle zijden even lang zijn. Je geeft de linkerbovenhoek en de zijlengte op:

square(x, y, zijde);
square(100, 100, 80);  // Vierkant van 80×80

rect() — Rechthoek

Een rechthoek heeft aparte breedte en hoogte. Net als square() is (x, y) de linkerbovenhoek:

rect(x, y, breedte, hoogte);
rect(100, 100, 200, 150);

rectMode(CENTER) — Middelpunt als ankerpunt

Wil je liever het midden van de rechthoek als coördinaat gebruiken? Stel dan rectMode(CENTER) in via setup():

function setup() {
  createCanvas(400, 400);
  rectMode(CENTER);  // Coördinaat = midden van de rechthoek
}

function draw() {
  rect(200, 200, 100, 60);  // Middelpunt op (200,200)
}

triangle() — Driehoek

Een driehoek heeft drie hoekpunten, elk met een x- en y-coördinaat:

triangle(x1, y1, x2, y2, x3, y3);
triangle(200, 50, 100, 200, 300, 200);

Coördinaten

Het canvas gebruikt een coördinatensysteem waarbij:

  • (0,0) is linksboven
  • X gaat naar rechts
  • Y gaat naar beneden

Dit is anders dan wiskunde waar Y omhoog gaat!

Vormen Combineren

Je kunt meerdere vormen tekenen in één draw() functie:

function draw() {
  background(245);
  ellipse(200, 200, 100, 100);  // Cirkel
  rect(150, 150, 100, 100);     // Rechthoek
  line(0, 0, 400, 400);         // Lijn
}

De volgorde bepaalt wat bovenop ligt - laatste vorm is bovenop.

arc() en quad()

arc() — Boog of Taartpunt

Een boog is een stuk van een ellips. Je geeft het middelpunt, de maten en twee hoeken op (in radialen):

// arc(x, y, breedte, hoogte, startHoek, eindHoek)
arc(200, 200, 150, 150, 0, PI);          // Halve cirkel (onderkant)
arc(200, 200, 150, 150, 0, PI, PIE);     // Taartpunt
arc(200, 200, 150, 150, 0, PI, CHORD);   // Gesloten koorde

Opties voor het sluittype: OPEN (standaard), CHORD (rechte lijn tussen uiteinden), PIE (taartpunt).

quad() — Vierhoek

Een vierhoek met vier hoekpunten — niet noodzakelijk een rechthoek:

// quad(x1, y1, x2, y2, x3, y3, x4, y4)
quad(100, 50, 300, 20, 350, 200, 80, 230);

Eigen vormen met beginShape()

Met beginShape() en vertex() teken je elke veelhoek die je wilt:

beginShape();
  vertex(50, 150);
  vertex(100, 50);   // Piek 1
  vertex(150, 120);
  vertex(200, 40);   // Piek 2
  vertex(300, 150);
endShape(CLOSE);     // CLOSE sluit de vorm automatisch

LINES en POINTS als argument

// Elke twee vertices vormen een lijn
beginShape(LINES);
  vertex(50, 50);  vertex(200, 50);
  vertex(50, 100); vertex(200, 100);
endShape();

// Enkel punten tekenen
beginShape(POINTS);
  vertex(100, 100);
  vertex(200, 150);
  vertex(300, 100);
endShape();

Vloeiende curves

Voor vloeiende lijnen door meerdere punten gebruik je beginShape() met splineVertex():

beginShape();
splineVertex(80, 260);
splineVertex(180, 120);
splineVertex(320, 300);
splineVertex(460, 140);
splineVertex(540, 250);
endShape();

Bézier en kwadratische curves

Je kunt ook Bézier-curves en kwadratische curves mixen binnen één vorm:

beginShape();
vertex(60, 220);
bezierVertex(140, 40, 260, 40, 320, 220);
quadraticVertex(430, 360, 540, 180);
endShape();
