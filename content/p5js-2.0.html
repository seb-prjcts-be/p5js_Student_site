p5.js 2.0 introduceert veel nieuwe features die het nog krachtiger maken. Leer wat er nieuw is en hoe je het gebruikt.

Async Assets Laden

In p5.js 2.0 kun je assets asynchroon laden met async/await:

let img;

async function setup() {
  createCanvas(800, 600);
  img = await loadImage('afbeelding.jpg');
}

function draw() {
  image(img, 0, 0);
}

Dit maakt het makkelijker om meerdere assets te laden en fouten af te handelen.

Nieuwe Typografie Features

p5.js 2.0 heeft verbeterde tekst rendering:

  • Betere font ondersteuning
  • Meer controle over tekst layout
  • Verbeterde performance

Curves en Splines

Nieuwe functies voor vloeiende curves:

  • bezier() - Bézier curves
  • curve() - Catmull-Rom splines
  • quadraticVertex() - Kwadratische curves

Pointer Events

Nieuwe events voor touch en pointer input:

  • pointerX, pointerY - Werkt voor muis én touch
  • pointerPressed() - Event voor pointer down
  • pointerMoved() - Event voor pointer beweging
  • pointerReleased() - Event voor pointer up
