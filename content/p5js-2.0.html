In deze cursus werken we standaard met moderne p5.js. Dat betekent: assets laad je meestal asynchroon in async function setup(), je kunt pointer input gebruiken voor muis en touch, en je hebt sterke tools voor typografie, curves en WebGL. Oudere patronen zoals preload() kom je soms nog tegen in bestaande tutorials, maar ze zijn hier niet het vertrekpunt.

Wat moderne p5.js hier betekent

  • Assets laden doe je meestal met await loadImage(), await loadFont() of await loadShader() in setup()
  • Input kan via pointer events werken voor muis en touch tegelijk
  • Typografie en WebGL sluiten aan op een modernere browseromgeving
  • Oudere code blijft vaak leesbaar, maar is voor deze cursus niet de standaard schrijfwijze
let achtergrond;

async function setup() {
  achtergrond = await loadImage('achtergrond.jpg');
  createCanvas(600, 400);
}

function draw() {
  image(achtergrond, 0, 0, width, height);
}

Assets laden met async/await

Als je een afbeelding, font of shader nodig hebt, laad je die in moderne p5.js meestal rechtstreeks in setup(). Je sketch wacht dan netjes tot het bestand klaar is.

let img;
let lettertype;

async function setup() {
  [img, lettertype] = await Promise.all([
    loadImage('foto.jpg'),
    loadFont('mijn-font.ttf')
  ]);

  createCanvas(600, 400);
  textFont(lettertype);
}

function draw() {
  background(24);
  image(img, 0, 0, width, height);
  fill(255);
  textSize(28);
  text('Alles is geladen', 20, 40);
}

Oudere code die je nog kunt tegenkomen

let img;

function preload() {
  img = loadImage('foto.jpg');
}

function setup() {
  createCanvas(600, 400);
}

preload() is niet fout. Het is vooral handig om oudere voorbeelden te kunnen lezen. In deze cursus schrijven we nieuwe oefeningen liever met async/await.

Pointer input voor muis en touch

Met pointer input hoef je minder apart te denken in muis tegenover touchscreen. Dat is handig als een sketch op laptop én tablet moet werken.

Pointer variabelen

function draw() {
  background(220);

  fill(255, 120, 0);
  circle(pointerX, pointerY, 60);

  if (pointerIsDown) {
    fill(0, 180, 120);
    circle(pointerX, pointerY, 100);
  }
}

Pointer events

function pointerPressed() {
  fill(random(255), random(255), random(255));
  circle(pointerX, pointerY, 40);
}

function pointerMoved() {
  stroke(0, 140, 255);
  strokeWeight(3);
  line(ppointerX, ppointerY, pointerX, pointerY);
}

Je zult nog vaak mousePressed(), mouseDragged() en touchStarted() zien in bestaand lesmateriaal. Voor nieuwe oefeningen kun je pointer events als eerste keuze nemen wanneer je één invoermodel wilt voor meerdere toestellen.

Curves en shapes

Curves zijn niet nieuw als idee, maar ze passen goed in een moderne p5.js-workflow waarin je snelle schetsen combineert met vloeiende vormen en interactieve input.

Catmull-Rom met curveVertex()

beginShape();
curveVertex(80, 260);
curveVertex(80, 260);
curveVertex(180, 120);
curveVertex(320, 300);
curveVertex(460, 140);
curveVertex(540, 250);
curveVertex(540, 250);
endShape();

Bézier en quadratische curves

beginShape();

Typografie als volwaardig materiaal

Tekst is in moderne p5.js niet alleen een label. Je kunt fonts laden, uitlijning fijn regelen en typografie echt als beeldmateriaal inzetten.

let titelFont;

async function setup() {
  titelFont = await loadFont('mijn-lettertype.ttf');
  createCanvas(600, 400);
  textFont(titelFont);
}

function draw() {
  background(245);
  fill(30);
  textSize(56);
  textAlign(CENTER, CENTER);
  text('Hallo typografie', width / 2, height / 2);
}

Nuttige teksttools

textLeading(36);
textAlign(LEFT, TOP);

let woord = 'Lab44';
let breedte = textWidth(woord);
text(woord, width / 2 - breedte / 2, 120);

WebGL en shaders

Ook in 3D kun je moderne p5.js inzetten als standaardbasis. Zeker bij shaders is async/await handig, omdat bestanden eerst moeten laden.

let mijnShader;

async function setup() {
  createCanvas(600, 400, WEBGL);
  mijnShader = await loadShader('shader.vert', 'shader.frag');
}

function draw() {
  shader(mijnShader);
  mijnShader.setUniform('uTime', millis() / 1000);
  rect(-width / 2, -height / 2, width, height);
}

Je hoeft niet meteen met shaders te starten. Het belangrijkste is dat je weet dat moderne p5.js dit soort workflows netjes ondersteunt.
