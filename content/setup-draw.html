Elke p5.js sketch begint met twee belangrijke functies: setup() en draw(). Deze vormen de basis van je creatie.

Kernpunten
  • De rol van setup() (één keer) en draw() (continue lus)
  • Een canvas aanmaken en de maten kiezen
  • Basis-schrijfregels van JavaScript, commentaar en constanten zoals PI
  • Framerate regelen met frameRate() en de lus pauzeren met noLoop()

setup() - De Start

setup() wordt één keer uitgevoerd wanneer je sketch start. Hierin zet je de basis op:

function setup() {
  createCanvas(800, 600);  // Maak een canvas van 800x600 pixels
  background(245);          // Zet achtergrondkleur
}

Wat doe je in setup()?

  • Canvas aanmaken met createCanvas()
  • Begintoestand instellen
  • Assets laden (afbeeldingen, geluiden)

draw() - De Loop

draw() wordt continu uitgevoerd (meestal 60 keer per seconde). Hierin teken je je animatie:

function draw() {
  background(245);           // Wis het canvas
  ellipse(mouseX, mouseY, 50, 50);  // Teken cirkel op muispositie
}

Belangrijk

  • draw() loopt automatisch door
  • Zonder background() zie je trails van vorige frames
  • Met background() maak je een schone achtergrond

Canvasgrootte

Het canvas is je tekengebied. Je kunt verschillende formaten maken:

createCanvas(400, 400);   // Vierkant
createCanvas(800, 600);   // Landscape
createCanvas(600, 800);   // Portrait
createCanvas(windowWidth, windowHeight);  // Volledig scherm

Schrijfregels (syntaxis)

p5.js volgt de regels van JavaScript. Een paar belangrijke gewoontes:

  • Elke opdracht eindigt met een puntkomma ;
  • Functienamen gebruiken lowerCamelCase: createCanvas(), noStroke()
  • De code is hoofdlettergevoelig: background() werkt, Background() niet
  • Accolades { } bakenen het begin en einde van een functie af

Commentaar

Commentaar is tekst die niet als code wordt gelezen. Handig om aantekeningen te maken of tijdelijk code uit te zetten:

// Dit is een commentaarregel - wordt genegeerd door p5.js

/* Dit is een blok commentaar
   dat meerdere regels kan beslaan */

let x = 10;  // Variabele x instellen op 10

Gebruik in de browser Ctrl + / om een selectie in commentaar te zetten.

Constanten

Sommige vaste waarden zijn altijd in hoofdletters geschreven, zoals PI, TWO_PI, HALF_PI en WEBGL. Ze worden constanten genoemd omdat hun waarde nooit verandert.

// PI is altijd 3.14159...
ellipse(200, 200, PI * 10, PI * 10);

frameRate() en noLoop()

Standaard roept draw() zichzelf zo snel mogelijk aan (vaak ~60 keer per seconde). Je kunt de snelheid regelen:

function setup() {
  createCanvas(400, 400);
  frameRate(30);  // Begrens op 30 frames per seconde
}

function draw() {
  background(220);
  // Animatie hier
}

Met noLoop() stop je de loop - draw() wordt dan maar één keer uitgevoerd. Handig voor statische composities.

function setup() {
  createCanvas(400, 400);
  noLoop();  // Teken eenmalig
}
