Gooi een dobbelsteen, spring halverwege naar een hoekpunt, herhaal. Uit deze simpele regel ontstaat een van de mooiste fractals in de wiskunde: de Sierpinski-driehoek.

Wat is het Chaos Game?

Het Chaos Game is een algoritme dat met pure willekeur een perfect geometrisch patroon genereert. De regels zijn verrassend eenvoudig:

  1. Kies drie punten op je canvas die samen een driehoek vormen.
  2. Nummer de hoekpunten van 0 tot 2.
  3. Kies een willekeurig startpunt P ergens op het canvas.
  4. Maak een digitale dobbelsteen met drie waarden: floor(random(3)).
  5. Gooi de dobbelsteen en beweeg P halverwege naar het gekozen hoekpunt.
  6. Teken een punt op de nieuwe positie van P.
  7. Herhaal stap 4-6 duizenden keren.

Na een paar duizend iteraties verschijnt er een herkenbaar patroon: de Sierpinski-driehoek.

Waarom werkt dit?

Het lijkt onmogelijk: willekeurige keuzes die een perfecte fractal opleveren. Toch is het logisch als je erbij stilstaat.

De halverwege-regel zorgt ervoor dat punten steeds dichter bij de hoekpunten komen te liggen. Na een paar stappen belandt elk punt in een van de drie "zones" van de Sierpinski-driehoek. Het midden wordt nooit geraakt - precies het gat in het midden van de fractal.

Dit patroon is ook te maken met recursie, maar het Chaos Game laat zien dat toeval en structuur niet elkaars tegengestelden zijn.

De basiscode

Het kernalgoritme in p5.js:

// Drie hoekpunten van de driehoek
let hoeken = [
  { x: 300, y: 20 },
  { x: 20, y: 380 },
  { x: 580, y: 380 }
];

// Startpunt
let px = random(width);
let py = random(height);

// Elke frame: 100 nieuwe punten
for (let i = 0; i < 100; i++) {
  let keuze = floor(random(3)); // dobbelsteen: 0, 1 of 2
  px = (px + hoeken[keuze].x) / 2; // halverwege x
  py = (py + hoeken[keuze].y) / 2; // halverwege y
  point(px, py);
}

De floor(random(3)) is je digitale dobbelsteen: hij geeft willekeurig 0, 1 of 2 terug. Met (px + hoek.x) / 2 spring je naar het midden tussen je huidige positie en het gekozen hoekpunt.

Variaties

Het Chaos Game is niet beperkt tot driehoeken. Door de regels aan te passen krijg je verrassende resultaten:

  • Andere ratio: in plaats van halverwege (0.5), probeer 1/3 of 2/3. De fractal verandert van vorm.
  • Meer hoekpunten: met 4 punten (vierkant) en de restrictie dat je niet twee keer naar hetzelfde hoekpunt mag, ontstaat een ander patroon.
  • Kleuren per hoekpunt: geef elk hoekpunt een eigen kleur en kleur het punt op basis van de laatste keuze. De structuur wordt nog duidelijker.
  • Grootte variatie: maak de puntgrootte afhankelijk van het iteratienummer voor een diepte-effect.

Oefening

Probeer de volgende aanpassingen in de editor hieronder:

  • Verander de ratio van 0.5 naar 0.33 - wat gebeurt er met het patroon?

  • Geef elk hoekpunt een eigen kleur met stroke() op basis van de dobbelsteenwaarde.
