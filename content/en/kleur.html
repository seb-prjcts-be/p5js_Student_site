Color brings your sketches to life. p5.js offers several ways to use colors.

Color functions

You always set the color before you draw the shape. p5.js reads code from top to bottom:

  • background(color) - background color of the canvas
  • fill(color) - fill color for shapes
  • stroke(color) - line color (outline)
  • strokeWeight(thickness) - thickness of the outline in pixels
  • noFill() - no fill (transparent interior)
  • noStroke() - no outline
stroke(255, 0, 0);       // Red outline
strokeWeight(3);          // Line thickness 3 pixels
fill(0, 200, 255);        // Light blue fill
circle(200, 200, 100);

noStroke();               // No outline for next shapes
fill(255, 150, 0);
rect(100, 50, 80, 80);

Color values

Colors can be specified in different ways:

Grayscale (0-255)

fill(255);        // White
fill(128);        // Gray
fill(0);          // Black

RGB (Red, Green, Blue)

fill(255, 0, 0);      // Red
fill(0, 255, 0);      // Green
fill(0, 0, 255);      // Blue
fill(255, 255, 0);    // Yellow

RGBA (With Transparency)

fill(255, 0, 0, 128);  // Half transparent red
fill(0, 0, 255, 200);  // Nearly opaque blue

Color names

You can also use CSS color names as text:

fill('red');
rect(100, 100, 200, 100);

fill('white');
square(100, 200, 100);

Other color notations

Besides RGB you can also specify colors as a hexadecimal code (like in CSS) or as a CSS name:

// Hexadecimal code (# + 6 digits)
fill('#FF6600');    // Orange
stroke('#003399'); // Dark blue

// CSS color names
fill('tomato');
fill('steelblue');
fill('gold');

You can find hexadecimal codes using color pickers in Photoshop, Figma, or on websites like coolors.co.

Random colors

With random(255) you generate a random color value between 0 and 255:

// Random color
fill(random(255), random(255), random(255));

// Random shade of gray
let gray = random(255);
fill(gray);

// Random color with fixed transparency
fill(random(255), random(255), random(255), 180);

Tips and pitfalls

background() with transparency

A powerful creative trick: use background() with a low alpha value in draw(). Instead of completely clearing the canvas, you lay a semi-transparent layer over the previous frame. The result: a trailing or ghost effect.

function draw() {
  background(0, 15);  // Black with low alpha - trail effect
  fill(255);
  circle(mouseX, mouseY, 30);
}

The lower the number, the longer the trail lingers. Try values between 5 and 30.

Lines and points have no fill

A common beginner mistake: calling fill() before a line() or point(). That has no effect - lines and points only respond to stroke() and strokeWeight().

stroke(255, 0, 0);
strokeWeight(3);
line(10, 10, 200, 200);  // Red line of 3 pixels

// fill() does nothing here:
fill(0, 255, 0);
point(100, 100);  // Only stroke counts

Very thin lines with sub-pixel values

strokeWeight() also accepts decimal values. With strokeWeight(0.25) or strokeWeight(0.5) you create hair-thin lines - ideal for fine grids or generative patterns.

strokeWeight(0.25);
for (let x = 0; x < width; x += 4) {
  line(x, 0, x, height);  // Hair-thin grid
}

Color mode

By default p5.js uses RGB. You can also use HSB (Hue, Saturation, Brightness):

colorMode(HSB);
fill(0, 100, 100);    // Red (hue=0)
fill(120, 100, 100);  // Green (hue=120)
fill(240, 100, 100);  // Blue (hue=240)

HSB is useful for color wheels and smooth color transitions.

Color Variables

You can store colors in variables:

let red = color(255, 0, 0);
let blue = color(0, 0, 255);

function draw() {
  fill(red);
  ellipse(100, 200, 50, 50);

  fill(blue);
  ellipse(300, 200, 50, 50);
}
