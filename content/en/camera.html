With a single function p5.js can access your webcam: createCapture(). That opens the door to real-time interaction, live effects and artworks that respond to whatever happens in front of the screen.

What you will learn
  • Webcam access through createCapture(VIDEO)
  • Hiding the video element with v.hide() so only the canvas shows
  • Applying filters: GRAY, INVERT, POSTERIZE, BLUR
  • Combining the webcam with other p5.js functions for interactive sketches
Watch out

Make sure your laptop's camera works and that no cover is slid over it. Test your camera by typing "Camera" in your Windows search bar. The browser will ask for permission the first time you use the webcam.

A simple webcam sketch

With createCapture(VIDEO) you get a video object that continuously delivers new frames from your webcam. You draw it on the canvas with image(), just like a regular image. Hide the separate HTML video element that p5 creates with v.hide(), otherwise you see the feed twice.

let v;

function setup() {
  createCanvas(400, 400);
  v = createCapture(VIDEO);
  v.hide(); // otherwise you see the same video twice
}

function draw() {
  image(v, 0, 0);
}

Interacting with the webcam

You can combine webcam data with other p5.js functions to build interactive applications. For example: draw an object that reacts to motion in the image, or add a crazy noise effect.

Effects and filters

Filters can be applied to images, videos and basically to your entire canvas. Whether you want a vintage look or real-time effects on a webcam feed: one line of code can go a long way.

p5.js offers a set of ready-made filters via the filter() function. Common constants are GRAY, INVERT, POSTERIZE and BLUR.

Filter on the webcam

let v;

function setup() {
  createCanvas(400, 400);
  v = createCapture(VIDEO);
  v.hide();
}

function draw() {
  image(v, 0, 0);
  filter(INVERT);
}

Try POSTERIZE(3) or BLUR(5) with a parameter, or stack several filters for unexpected combinations.
