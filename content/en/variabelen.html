Variables store values that you can use later. They are essential for dynamic and interactive sketches.

Creating Variables

In JavaScript you use let or const to create variables:

let x = 100;        // Can change
const y = 200;      // Stays constant
let diameter = 50;  // Diameter of circle

When to use which?

  • let - For values that can change
  • const - For values that don't change

Variable types

JavaScript has different kinds of values. In p5.js it usually doesn't matter much, but it's good to know the types:

Integer (whole number)

let age = 25;
let x = -5;

Float (decimal number)

Use a dot for decimals, not a comma:

let speed = 2.5;
let weight = 68.5;

String (text)

Text is always placed between quotation marks — single or double:

let name = "Jan";
let birthdate = "10/04/1984";
let message = 'Hello World';

String concatenation

You can join text and variables together with a + sign:

let name = "Els";
let x = 100;
let sentence = name + " has " + x + " apples.";
console.log(sentence);
// → Els has 100 apples.

Note: the total number of quotation marks must always be even.

Boolean (true/false)

A boolean has only two possible values: true or false. With an exclamation mark ! you can flip the value:

let isActive = true;
let isPaused = false;

isActive = !isActive;  // Now false

Numbers

let x = 100;
let speed = 2.5;
let percentage = 0.75;

Using Variables

Variables make your code flexible:

let x = 100;
let y = 200;
let diameter = 50;

function draw() {
  ellipse(x, y, diameter, diameter);

  // Change values
  x += 1;  // Move to the right
}

Where does a variable live?

Not every variable is available everywhere. It depends on where you declare it:

  • Outside all functions → global: available throughout your entire sketch
  • Inside a function → local: exists only while that function is running
let globalX = 0;  // Available in setup(), draw(), and everywhere

function draw() {
  let localY = 100;  // Only exists inside draw()
  globalX += 1;      // Works — globalX is from outside

  // localY is still available here...
}
// ...but not here anymore

// globalX still exists

In p5.js, you always declare variables that you need in multiple functions — like positions, speeds, or counters — outside setup() and draw(). That way they keep their value from frame to frame.

let x = 0;          // Remembered between frames

function setup() {
  createCanvas(600, 400);
}

function draw() {
  background(245);
  circle(x, 200, 40);
  x += 2;           // x grows each frame
}

Built-in system variables

p5.js manages a number of variables automatically for you. You don't need to create them yourself — they're always available:

width and height

The width and height of the canvas that you set in createCanvas():

function draw() {
  let centerX = width / 2;   // Horizontal center of the canvas
  let centerY = height / 2;  // Vertical center
  circle(centerX, centerY, 100);
}

windowWidth and windowHeight

The width and height of the browser window (not the canvas). Useful for creating a canvas that fills the entire window:

function setup() {
  createCanvas(windowWidth, windowHeight);
}

// Adjust when the window is resized:
function windowResized() {
  resizeCanvas(windowWidth, windowHeight);
}

Calculations

With variables you can do math: addition, subtraction, multiplication, division. The four basic operators, the modulo %, and useful functions like map() and dist() are covered in detail on the Calculations page.

let centerX = width / 2;
let centerY = height / 2;
let red = map(mouseX, 0, width, 0, 255);  // scale mouse to color
