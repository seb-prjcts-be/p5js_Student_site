A flowfield is an invisible current across the canvas. At every point noise() decides an angle, and lines, particles or shapes follow those angles. The result: coherent waves, swirls and currents that feel alive.

What you will learn
  • What a flowfield is: a grid of angles generated with noise()
  • map() to translate noise values (0-1) into angles (0-360)
  • translate() and rotate() to rotate each cell locally
  • push() and pop() to isolate transformations
  • Animation by shifting noise with a t variable

What is a flowfield?

A flowfield is a region that uses a noise() function to generate coherent drawings or animations. Picture it as an invisible wind blowing across the canvas, where at every point the wind has a different direction. We use map() to translate the values between 0 and 1 into something useful - for instance a rotation from 0 to 360.

Basic example

We walk through a grid of cells with a double for loop. For each cell we request a noise value, convert it to an angle, and draw a small cross rotated to that angle.

let cell = 15;
let s = 300;
let t = 0;

function setup() {
  createCanvas(400, 400);
  angleMode(DEGREES);
  rectMode(CENTER);
}

function draw() {
  background(245);
  translate(60, 60);
  for (let y = cell; y < s - 10; y += cell) {
    for (let x = cell; x < s - 20; x += cell) {
      let n = map(noise(x * 0.01 + t, y * 0.01 + t), 0, 1, 0, 360);
      push();
        translate(x + cell / 2, y + cell / 2);
        rotate(n);
        line(0, cell, cell, cell);
        line(cell / 2, cell / 2, cell / 2, cell);
      pop();
    }
  }
  t += 0.008;
}

Experiment yourself

Tweak the cell size, or the step t += 0.008, and watch the current change.

Variations

  • Replace the crosses with short lines or arrows that flow with the current.
  • Let particles at random starting positions shift slightly every frame along the local angle.
  • Use strokeWeight() to vary line thickness with the noise value.
  • Combine with a colour scale: map the angle to a hue in HSB mode as well.

Flowfields work beautifully with boids-style agents: give each particle a position and let it accelerate along the local flowfield angle every frame.
