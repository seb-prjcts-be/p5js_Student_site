Mouse interaction makes your sketches responsive. p5.js has built-in variables and functions for mouse input.

Mouse Position

p5.js has built-in variables for mouse position:

  • mouseX - X position of the mouse
  • mouseY - Y position of the mouse
  • pmouseX - Previous X position
  • pmouseY - Previous Y position

Example

function draw() {
  background(245);
  ellipse(mouseX, mouseY, 50, 50);
}

Mouse Events

You can respond to mouse clicks and movements:

mousePressed()

function mousePressed() {
  // Runs on mouse click
  fill(random(255), random(255), random(255));
}

mouseDragged()

function mouseDragged() {
  // Runs while dragging
  ellipse(mouseX, mouseY, 20, 20);
}

mouseReleased()

function mouseReleased() {
  // Runs on release
  background(245);
}

Pointer input: mouse and touch

Pointer variables work on both mouse and touchscreen. Handy when a sketch needs to work on laptop and tablet.

Pointer variables

  • pointerX / pointerY — current position
  • ppointerX / ppointerY — previous position
  • pointerIsDown — true while pressed
function draw() {
  background(245);
  fill(255, 120, 0);
  circle(pointerX, pointerY, 60);

  if (pointerIsDown) {
    fill(0, 180, 120);
    circle(pointerX, pointerY, 100);
  }
}

Pointer events

function pointerPressed() {
  fill(random(255), random(255), random(255));
  circle(pointerX, pointerY, 40);
}

function pointerMoved() {
  stroke(0, 140, 255);
  strokeWeight(3);
  line(ppointerX, ppointerY, pointerX, pointerY);
}

You'll still encounter mousePressed() and touchStarted() a lot in existing learning materials. For new sketches that need to work on multiple devices, pointer events are the first choice.

Creating Trails

With pmouseX and pmouseY you can create trails:

function draw() {
  // Draw line from previous to current mouse position
  line(pmouseX, pmouseY, mouseX, mouseY);
}
