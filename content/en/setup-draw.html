Every p5.js sketch starts with two important functions: setup() and draw(). These form the foundation of your creation.

What you will learn
  • The role of setup() (runs once) and draw() (continuous loop)
  • Creating a canvas and choosing its size
  • Basic JavaScript syntax rules, comments, and constants like PI
  • Controlling the framerate with frameRate() and pausing the loop with noLoop()

setup() - The Start

setup() is executed once when your sketch starts. This is where you set up the basics:

function setup() {
  createCanvas(800, 600);  // Create a canvas of 800x600 pixels
  background(245);          // Set background color
}

What do you do in setup()?

  • Create a canvas with createCanvas()
  • Set the initial state
  • Load assets (images, sounds)

draw() - The Loop

draw() runs continuously (usually 60 times per second). This is where you draw your animation:

function draw() {
  background(245);           // Clear the canvas
  ellipse(mouseX, mouseY, 50, 50);  // Draw circle at mouse position
}

Important

  • draw() loops automatically
  • Without background() you'll see trails from previous frames
  • With background() you create a clean background

Canvas size

The canvas is your drawing area. You can create different sizes:

createCanvas(400, 400);   // Square
createCanvas(800, 600);   // Landscape
createCanvas(600, 800);   // Portrait
createCanvas(windowWidth, windowHeight);  // Full screen

Syntax rules

p5.js follows the rules of JavaScript. A few important conventions:

  • Every statement ends with a semicolon ;
  • Function names use lowerCamelCase: createCanvas(), noStroke()
  • Code is case-sensitive: background() works, Background() does not
  • Curly braces { } mark the beginning and end of a function

Comments

Comments are text that is not read as code. Useful for making notes or temporarily disabling code:

// This is a comment line - ignored by p5.js

/* This is a block comment
   that can span multiple lines */

let x = 10;  // Set variable x to 10

In the browser, use Ctrl + / to comment out a selection.

Constants

Some fixed values are always written in uppercase, such as PI, TWO_PI, HALF_PI, and WEBGL. They are called constants because their value never changes.

// PI is always 3.14159...
ellipse(200, 200, PI * 10, PI * 10);

frameRate() and noLoop()

By default, draw() calls itself as fast as possible (often ~60 times per second). You can control the speed:

function setup() {
  createCanvas(400, 400);
  frameRate(30);  // Limit to 30 frames per second
}

function draw() {
  background(220);
  // Animation here
}

With noLoop() you stop the loop - draw() is then executed only once. Useful for static compositions.

function setup() {
  createCanvas(400, 400);
  noLoop();  // Draw once
}
