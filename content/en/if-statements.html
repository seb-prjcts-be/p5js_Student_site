With if-statements you make decisions in your code. Your code can do different things depending on conditions.

What you will learn
  • The structure of if, else if, and else
  • Comparison operators: ==, !=, >, <, >=, <=
  • Combining with logical operators: &&, ||, !
  • Short notation without braces for a single statement
  • Using a boolean variable directly as the condition

Basic If-Statement

An if-statement checks a condition:

if (condition) {
  // Code that runs if the condition is true
}

Example

function draw() {
  if (mouseX > width/2) {
    fill(255, 0, 0);  // Red if mouse is on the right
  } else {
    fill(0, 0, 255);  // Blue if mouse is on the left
  }
  ellipse(200, 200, 100, 100);
}

Comparisons

You can make different comparisons:

  • == - Equal to
  • != - Not equal to
  • > - Greater than
  • < - Less than
  • >= - Greater than or equal to
  • <= - Less than or equal to

Examples

if (x > 100) { }
if (y < 200) { }
if (diameter == 50) { }
if (speed != 0) { }

Else and Else If

You can check multiple conditions:

if (x < 100) {
  fill(255, 0, 0);  // Red
} else if (x < 200) {
  fill(0, 255, 0);  // Green
} else {
  fill(0, 0, 255);  // Blue
}

Logical Operators

Combine conditions with && (and) and || (or), or reverse a condition with ! (not):

if (x > 100 && x < 300) {
  // x is between 100 and 300
}

if (x < 50 || x > 350) {
  // x is less than 50 OR greater than 350
}

// ! reverses a boolean
if (!mouseIsPressed) {
  background(245);  // Only clear when mouse is NOT pressed
}

Short notation

If you only want to execute one statement, you can omit the braces:

// With braces (standard)
if (x > width) {
  x = 0;
}

// Short notation (one statement)
if (x > width) x = 0;

Boolean as condition

If a variable is already a boolean (true/false), you don't need to compare:

let isStudent = true;

// Explicit
if (isStudent == true) {
  fill(0, 255, 0);
}

// Shorter - same effect
if (isStudent) {
  fill(0, 255, 0);
}

// Reversed with !
if (!isStudent) {
  fill(255, 0, 0);  // If NOT a student
}
