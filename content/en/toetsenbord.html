With keyboard input you can control your sketches. p5.js has functions to detect key presses.

What you will learn
  • Catching key presses with the event function keyPressed()
  • Reading the pressed key via key (character) or keyCode (number)
  • Recognising special keys: UP_ARROW, ENTER, BACKSPACE, ...
  • Reacting continuously with keyIsPressed inside draw()
  • Catching upper- and lowercase together, e.g. key === 'r' || key === 'R'

keyPressed()

The keyPressed() function is called when a key is pressed:

let colour = 0;

function keyPressed() {
  if (key === 'r' || key === 'R') {
    colour = color(255, 0, 0);  // Red
  } else if (key === 'g' || key === 'G') {
    colour = color(0, 255, 0);  // Green
  }
}

function draw() {
  fill(colour);
  ellipse(200, 200, 100, 100);
}

Key Variables

p5.js has variables for keys:

  • key - The pressed key (as a character)
  • keyCode - For special keys

keyIsPressed

You can check if a key is being held down:

function draw() {
  if (keyIsPressed) {
    if (key === 'w' || key === 'W') {
      y -= 2;  // Move up
    }
  }
}
