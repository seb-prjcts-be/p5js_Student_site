Strudel is a live-coding music instrument in your browser. You write short lines of text, press Ctrl + Enter, and you hear music instantly. It works as a drum machine, synthesizer, and sequencer all at once, but without complicated settings.

Every line of code you write is a pattern. Strudel divides rhythm and time mathematically, so everything stays in sync. More info and the web editor can be found at strudel.cc.

Tempo

Strudel counts in cycles per minute (cpm). 1 cycle equals 1 bar (4 beats). That's why you start almost every project with a tempo:

setcpm(90/4)

That feels roughly like 90 BPM. Guidelines:

  • Slow/spacious: setcpm(50/4)
  • Pop/hip-hop: setcpm(90/4)
  • Dance/fast: setcpm(130/4)

How do you play?

  • Ctrl + Enter: play or update the code
  • Ctrl + .: stop the sound

You usually need to click somewhere in the editor first so the browser allows audio.

With sample examples, the first tick may still be silent while the sound is loading. Just let the pattern run or press play again.
