AI tools come in all shapes and sizes. From playgrounds to test models to editors that code alongside you - this is an overview of the most useful tools for students and makers.

What is an API?

An API (Application Programming Interface) is a way to let two software systems talk to each other. You send a question, and you get an answer back - without knowing how it works on the other side.

Think of it like a restaurant: you order (the request), the kitchen prepares it (the processing), and the waiter brings the result (the response). You don't need to know how the kitchen works.

Almost all AI tools work via an API. When you write code that talks to an AI model, you send a request to such an API and process the response in your own application.

Memory and context

AI models start every conversation blank - they remember nothing from previous sessions. That sounds like a limitation, but there are smart ways to work around it:

Provide context

You can always give an AI model extra information in your prompt. Paste in some text, share a link, or describe your project. The more relevant context you provide, the better the answer.

CLAUDE.md — memory for your project

Tools like Claude Code use a CLAUDE.md file as long-term memory. You describe your project, preferences, and rules in it. At the start of every session, the AI reads that file and immediately knows what you're working on.

You can apply this principle anywhere: create a text file with context about your project and paste it at the beginning of each new conversation.

Skills and instructions

Many AI tools support custom instructions or skills - reusable instructions that you save. This way you don't have to explain what you want every time. Think of it as a recipe that the AI can follow again and again.

Knowledge cutoff

Models have a knowledge cutoff (cutoff date): they don't automatically know about recent events. But many tools can now search the web, and you can always provide current information in your prompt.

AI Labs and Playgrounds

The major AI companies offer their own playgrounds: environments where you can test models directly without programming.

Google AI Studio

Google AI Studio is Google's free playground for Gemini models. You can test text, images, and code there, and create your own API key.

  • Try multimodal prompts: include an image with your question
  • Check out the tokenizer: see how your text is split up
  • Free to use for experimentation

OpenAI Platform

OpenAI Platform is the official playground of OpenAI (makers of ChatGPT). You can test GPT models here, explore fine-tuning, and analyze API requests.

Anthropic Console

Anthropic Console is the playground of Anthropic (makers of Claude). Similar to the others - handy for exploring Claude's strengths, such as processing longer documents and accurate reasoning.

NotebookLM

NotebookLM (from Google) is a tool that combines AI with your own sources. You upload documents, PDFs, or URLs, and then you can ask questions about them - the model only answers based on what you've added.

Very useful for:

  • Summarizing and querying a long PDF
  • Merging notes from different sources
  • Generating an "audio overview": a simulated podcast conversation about your sources

NotebookLM hallucinates much less than regular chatbots, because it's bound to the documents you've provided.

Chrome browser tips

Your browser has become a powerful AI starting point:

  • Gemini in Chrome - you can ask questions directly in the address bar by typing @gemini
  • Extensions - tools like Monica, Merlin, or Sider add an AI sidebar to every page
  • Reading and summary tools - select text on a page and directly request a summary

Other useful AI tools

Gamma

Gamma creates presentations, documents, and websites based on a text prompt. Handy as a starting point - customize to your own style afterwards.

ElevenLabs

ElevenLabs offers high-quality text-to-speech. Generate voices in dozens of languages and styles. Use it for videos, prototypes, or audio experiments.

Napkin

Napkin automatically converts text into diagrams and infographics. Handy for visualizing processes or ideas.

Eightify

Eightify is a Chrome extension that summarizes YouTube videos. Get the key points of a long video in seconds.

Programming with AI

AI is also changing how we write code. There are tools that sit right in your code editor and suggest, explain, and complete code.

GitHub Copilot (in VSCode)

Copilot suggests code as you type - based on the context of your file. Available as an extension in VSCode. Free for students via the GitHub Student Developer Pack.

Cursor

Cursor is a separate code editor built around AI. You can have it write complete code blocks, explain or refactor using plain language. Works well for larger projects.

Tips for AI-assisted programming

  • Describe what you want to achieve, not how - let the AI choose the implementation
  • Always review generated code - AI makes mistakes
  • Use AI for explanations: "Explain what this function does" works excellently
  • Small steps: let AI write one function at a time, not an entire project

Keeping up with the AI landscape

The AI field moves fast. Here are reliable sources to stay current:

  • Hugging Face - open-source models and papers, the hub of the AI research community
  • FutureTools.io - overview of new AI tools, sorted by category
  • Matt Wolfe (YouTube) - weekly videos about new AI tools and developments, accessibly explained
  • AI Explained (YouTube) - more in-depth videos about how models work

You're not expected to follow everything. Pick one or two sources that match your interests.
