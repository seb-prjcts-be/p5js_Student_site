Game of Life is a cellular automaton by John Conway. There are no players and no goal: just cells that live or die according to four rules.

Why this concept is so famous

Game of Life shows how complex behavior can emerge from almost nothing. With four simple rules you get patterns that pulse, move, collide, or disappear.

Think about it: when does a system feel "alive," even though nothing is actually living?

What you will learn here

  • Working with grids and neighbor relationships
  • Calculating states across successive generations
  • Seeing how local rules create global patterns
  • Building simulations with two grids: current and next

The rules

Each cell is alive (1) or dead (0). Each generation changes based on its 8 neighbors:

  • Living cell with 2 or 3 neighbors - survives
  • Living cell with fewer than 2 neighbors - dies from isolation
  • Living cell with more than 3 neighbors - dies from overpopulation
  • Dead cell with exactly 3 neighbors - becomes alive

Those four rules are enough for still lifes, oscillators, and gliders.

Structure in p5.js

You work with two grids: the current one and the next one. This prevents a change mid-calculation from immediately affecting other cells.

for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let n = countNeighbors(grid, i, j);
    let cell = grid[i][j];
    if (cell === 1 && (n === 2 || n === 3)) next[i][j] = 1;
    else if (cell === 0 && n === 3)         next[i][j] = 1;
    else                                    next[i][j] = 0;
  }
}

[grid, next] = [next, grid];

Counting neighbors with wrapping

With wrapping, the right side connects to the left side and the top connects to the bottom:

function countNeighbors(g, x, y) {
  let sum = 0;
  for (let i = -1; i <= 1; i++) {
    for (let j = -1; j <= 1; j++) {
      if (i === 0 && j === 0) continue;
      sum += g[(x + i + cols) % cols][(y + j + rows) % rows];
    }
  }
  return sum;
}

Recognizing patterns

In Game of Life, shapes almost take on a personality of their own. Some stay still, some blink, some slide across the grid.

  • still life - no longer changes
  • oscillator - repeats itself after a few steps
  • glider - moves diagonally through the grid
