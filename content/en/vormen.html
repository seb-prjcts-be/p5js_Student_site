p5.js has built-in functions for basic shapes. With these functions you can quickly start drawing.

Basic elements: point and line

The simplest shapes in p5.js are a point and a line.

point() - Point

A point is the simplest shape, defined only by its x and y coordinates.

point(x, y);
point(100, 100);

line() - Line

A line has two endpoints, each with an x and y coordinate.

line(x1, y1, x2, y2);
line(0, 100, 400, 100);

Round shapes: circle() and ellipse()

circle() - Circle

Use a circle when width and height are equal. You specify the center point and the diameter:

circle(x, y, diameter);
circle(200, 200, 150);  // Circle with diameter 150

ellipse() - Oval

An ellipse has a separate width and height. If they're equal you also get a circle:

ellipse(x, y, width, height);
ellipse(200, 200, 100, 100);  // Circle via ellipse
ellipse(200, 200, 150, 100);  // Oval

Rectangular shapes: square() and rect()

square() - Square

Use a square when all sides are equal length. You specify the top-left corner and the side length:

square(x, y, side);
square(100, 100, 80);  // Square of 80x80

rect() - Rectangle

A rectangle has a separate width and height. Just like square(), (x, y) is the top-left corner:

rect(x, y, width, height);
rect(100, 100, 200, 150);

rectMode(CENTER) - Center as anchor point

Would you rather use the center of the rectangle as the coordinate? Then set rectMode(CENTER) in setup():

function setup() {
  createCanvas(400, 400);
  rectMode(CENTER);  // Coordinate = center of the rectangle
}

function draw() {
  rect(200, 200, 100, 60);  // Center at (200,200)
}

triangle() - Triangle

A triangle has three corner points, each with an x and y coordinate:

triangle(x1, y1, x2, y2, x3, y3);
triangle(200, 50, 100, 200, 300, 200);

Coordinates

The canvas uses a coordinate system where:

  • (0,0) is the top-left
  • X goes to the right
  • Y goes downward

This is different from math where Y goes upward!

Combining Shapes

You can draw multiple shapes in a single draw() function:

function draw() {
  background(245);
  ellipse(200, 200, 100, 100);  // Circle
  rect(150, 150, 100, 100);     // Rectangle
  line(0, 0, 400, 400);         // Line
}

The order determines what's on top - the last shape is on top.

arc() and quad()

arc() - Arc or Pie Slice

An arc is a portion of an ellipse. You specify the center, dimensions, and two angles (in radians):

// arc(x, y, width, height, startAngle, endAngle)
arc(200, 200, 150, 150, 0, PI);          // Half circle (bottom)
arc(200, 200, 150, 150, 0, PI, PIE);     // Pie slice
arc(200, 200, 150, 150, 0, PI, CHORD);   // Closed chord

Options for the close type: OPEN (default), CHORD (straight line between endpoints), PIE (pie slice).

quad() - Quadrilateral

A quadrilateral with four corner points - not necessarily a rectangle:

// quad(x1, y1, x2, y2, x3, y3, x4, y4)
quad(100, 50, 300, 20, 350, 200, 80, 230);

Custom shapes with beginShape()

With beginShape() and vertex() you can draw any polygon you want:

beginShape();
  vertex(50, 150);
  vertex(100, 50);   // Peak 1
  vertex(150, 120);
  vertex(200, 40);   // Peak 2
  vertex(300, 150);
endShape(CLOSE);     // CLOSE closes the shape automatically

LINES and POINTS as argument

// Every two vertices form a line
beginShape(LINES);
  vertex(50, 50);  vertex(200, 50);
  vertex(50, 100); vertex(200, 100);
endShape();

// Draw only points
beginShape(POINTS);
  vertex(100, 100);
  vertex(200, 150);
  vertex(300, 100);
endShape();

Smooth curves

For smooth lines through multiple points, use beginShape() with splineVertex():

beginShape();
splineVertex(80, 260);
splineVertex(180, 120);
splineVertex(320, 300);
splineVertex(460, 140);
splineVertex(540, 250);
endShape();

Bezier and quadratic curves

You can also mix Bezier curves and quadratic curves within a single shape:

beginShape();
vertex(60, 220);
bezierVertex(140, 40, 260, 40, 320, 220);
quadraticVertex(430, 360, 540, 180);
endShape();
