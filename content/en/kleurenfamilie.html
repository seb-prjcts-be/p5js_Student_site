Imagine: you're going to paint and you pick one paint pot as a starting point. After that you can only use colors that are close to it. This way you get a cohesive painting — not wild chaos. This code does the same thing, but digitally.

How does color work in p5.js?

Every color consists of three numbers between 0 and 255:

  • R — how much red
  • G — how much green
  • B — how much blue
fill(255, 0, 0)     // pure red
fill(200, 180, 190) // soft pinkish

If you pick all three channels completely at random, you get color chaos. The trick is to limit the randomness to a small window per channel.

The trick in three steps

Step 1 — pick a starting point per color channel

let startR = floor(random(0, 155));
let startG = floor(random(0, 155));
let startB = floor(random(0, 155));

Why up to 155 and not 255? Because we will add 100 later. 155 + 100 = 255 — this way we always stay within range.

Step 2 — define the range

// The range goes from startR to startR + 100
// Example: startR = 80 → red varies between 80 and 180

Step 3 — color each cell randomly, but within the range

fill(
  random(startR, startR + 100),
  random(startG, startG + 100),
  random(startB, startB + 100)
);

Each cell gets a different color — but never too different. They all stay in the same neighborhood.

Why is this clever?

Compare the two approaches:

// Without restriction — color chaos
fill(random(255), random(255), random(255));

// With restriction — color family
fill(
  random(startR, startR + 100),
  random(startG, startG + 100),
  random(startB, startB + 100)
);

With the restriction all colors stay in the same tonal neighborhood. This is exactly what textile designers do: they choose a palette of tones that are close together, so that each thread is slightly different but the fabric still looks harmonious. In the fashion industry this is called a colorway.

Drawing a grid with a color family

The double for-loop draws the grid: row by row, column by column. Each cell gets its own color, but always from the same family:

let startR, startG, startB;
let cellWidth = 60;
let cellHeight = 60;

function setup() {
  createCanvas(600, 400);
  noStroke();
  newColorFamily();
}

function newColorFamily() {
  startR = floor(random(0, 155));
  startG = floor(random(0, 155));
  startB = floor(random(0, 155));
}

function draw() {
  for (let y = 0; y < height; y += cellHeight) {
    for (let x = 0; x < width; x += cellWidth) {
      fill(
        random(startR, startR + 100),
        random(startG, startG + 100),
        random(startB, startB + 100)
      );
      rect(x, y, cellWidth, cellHeight);
    }
  }
  noLoop(); // drawing once is enough
}

function mousePressed() {
  newColorFamily();
  loop(); // draw again
}
