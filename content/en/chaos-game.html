Roll a die, jump halfway to a vertex, repeat. From this simple rule emerges one of the most beautiful fractals in mathematics: the Sierpinski triangle.

What is the Chaos Game?

The Chaos Game is an algorithm that generates a perfectly geometric pattern from pure randomness. The rules are surprisingly simple:

  1. Choose three points on your canvas that form a triangle.
  2. Number the vertices from 0 to 2.
  3. Choose a random starting point P somewhere on the canvas.
  4. Create a digital die with three values: floor(random(3)).
  5. Roll the die and move P halfway toward the chosen vertex.
  6. Draw a dot at the new position of P.
  7. Repeat steps 4-6 thousands of times.

After a few thousand iterations, a recognizable pattern appears: the Sierpinski triangle.

Why does this work?

It seems impossible: random choices producing a perfect fractal. Yet it makes sense when you think about it.

The halfway rule ensures that points keep getting closer to the vertices. After a few steps, every point lands in one of the three "zones" of the Sierpinski triangle. The center is never reached - exactly the gap in the middle of the fractal.

This pattern can also be created with recursion, but the Chaos Game shows that randomness and structure are not opposites.

The basic code

The core algorithm in p5.js:

// Three vertices of the triangle
let vertices = [
  { x: 300, y: 20 },
  { x: 20, y: 380 },
  { x: 580, y: 380 }
];

// Starting point
let px = random(width);
let py = random(height);

// Each frame: 100 new points
for (let i = 0; i < 100; i++) {
  let choice = floor(random(3)); // die: 0, 1 or 2
  px = (px + vertices[choice].x) / 2; // halfway x
  py = (py + vertices[choice].y) / 2; // halfway y
  point(px, py);
}

The floor(random(3)) is your digital die: it randomly returns 0, 1, or 2. With (px + vertex.x) / 2 you jump to the midpoint between your current position and the chosen vertex.

Variations

The Chaos Game is not limited to triangles. By adjusting the rules you get surprising results:

  • Different ratio: instead of halfway (0.5), try 1/3 or 2/3. The fractal changes shape.
  • More vertices: with 4 points (square) and the restriction that you can't jump to the same vertex twice in a row, a different pattern emerges.
  • Colors per vertex: give each vertex its own color and color the dot based on the last choice. The structure becomes even clearer.
  • Size variation: make the dot size depend on the iteration number for a depth effect.
