Loops repeat code. They're perfect for drawing multiple elements or iterating through data.

For Loop

The for loop is the most commonly used. It has three parts:

for (start; condition; step) {
  // Code that gets repeated
}

Example: 10 Circles

for (let i = 0; i < 10; i++) {
  ellipse(i * 50, 100, 40, 40);
}

This draws 10 circles next to each other.

Loop Variables

The loop variable (usually i) changes with each iteration:

for (let i = 0; i < 5; i++) {
  console.log(i);  // Prints: 0, 1, 2, 3, 4
}

Using in Drawing

for (let i = 0; i < 10; i++) {
  let x = i * 50;
  let y = 100;
  ellipse(x, y, 30, 30);
}

Nested loops

You can use loops inside loops for 2D patterns — one loop for the rows (y), one loop for the columns (x):

for (let x = 0; x < 10; x++) {
  for (let y = 0; y < 10; y++) {
    ellipse(x * 50, y * 50, 30, 30);
  }
}

This draws a grid of 10x10 circles.

Example with cell size

A handy technique is working with a cellSize variable instead of fixed distances:

let cellSize = 40;
let size = 400;

for (let j = 0; j < size; j += cellSize) {
  for (let i = 0; i < size; i += cellSize) {
    rect(i, j, cellSize);  // Square at position (i, j)
  }
}

The i represents all x-positions, the j represents all y-positions — this is a convention among programmers.

Artist Vera Molnar used exactly this principle (grid of cells) to control her drawing robot. Check the Vera Molnar page for examples.

While Loop

A while loop repeats as long as a condition is true:

let x = 0;
while (x < width) {
  line(x, 0, x, height);
  x += 20;
}

Watch out: make sure the condition eventually becomes false, otherwise you'll get an infinite loop!
