Circle packing is an algorithm that tries to place as many non-overlapping circles as possible in a space. Each circle grows from a random point until it touches another circle or the edge. The result is a dense, organic pattern that is both mathematically precise and visually appealing.

How does the algorithm work?

The principle is simple:

  1. Pick a random point on the canvas.
  2. Check if that point is far enough from all existing circles.
  3. If so: add a new small circle at that point.
  4. Each frame, let all growing circles get slightly larger.
  5. A circle stops growing as soon as it touches another circle or the edge.

You repeat steps 1-3 a few times each frame. Over time the space fills up so much that no new circles fit — then the packing is complete.

Detecting overlap with dist()

The core of the algorithm is checking whether two circles overlap. Two circles overlap when the distance between their centers is less than the sum of their radii:

// Do circle A and circle B overlap?
let distance = dist(a.x, a.y, b.x, b.y);
if (distance < a.r + b.r) {
  // They overlap
}

To maintain a safe margin, add a small buffer:

if (distance < a.r + b.r + 2) {
  // Too close — don't place
}

Storing a circle as an object

Each circle has a position, a radius, and a flag that tracks whether it is still growing:

let circles = [];

function addCircle(x, y) {
  circles.push({ x: x, y: y, r: 2, growing: true });
}

function drawAndGrow() {
  for (let c of circles) {
    circle(c.x, c.y, c.r * 2);

    if (c.growing) {
      c.r += 0.5;
      if (touchesEdge(c) || touchesOther(c)) {
        c.r -= 0.5; // step back
        c.growing = false;
      }
    }
  }
}

Checking edges and neighboring circles

function touchesEdge(c) {
  return c.x - c.r <= 0 || c.x + c.r >= width ||
         c.y - c.r <= 0 || c.y + c.r >= height;
}

function touchesOther(c) {
  for (let other of circles) {
    if (other === c) continue;
    if (dist(c.x, c.y, other.x, other.y) < c.r + other.r + 1) {
      return true;
    }
  }
  return false;
}

The check other === c is important: a circle should not consider itself as a neighbor.
