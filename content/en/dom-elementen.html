So far we drew everything onto the canvas. But what if you want buttons, sliders or text fields? That is where DOM (Document Object Model) elements come in: the regular HTML building blocks, but created from your p5.js code.

What you will learn

  • Creating buttons with createButton()
  • Creating and reading sliders with createSlider()
  • Reading text fields with createInput()
  • Positioning and styling with position() and style()
  • Adding interaction with mousePressed() events

What are DOM elements?

DOM elements are objects you can use to influence the structure and content of a web page. Think of buttons, sliders, text fields and much more. p5.js makes it easy to create them directly from your sketch.

Text field

// Text field with "First name" as placeholder
let firstName = createInput("First name");

// Read the value
print(firstName.value());

Slider

// Slider from 0 to 100, starting at 50, in steps of 2
let slider = createSlider(0, 100, 50, 2);

// Read the value
print(slider.value());

Button

// Button with the text "Click me"
let button = createButton("Click me");

Position and style

Besides the CSS file you can also set position and style of DOM elements directly from p5.js.

button.position(100, 100);
button.style('background-color', 'purple');

Interaction and events

DOM elements can respond to user actions like clicking or typing. The button below calls changeColor() when you click it.

button.mousePressed(changeColor);

function changeColor() {
  button.style('background-color', 'lime');
}

This leads us straight to the next chapter on functions: your own piece of code that you can hook up to an event.

All together - experiment

A slider drives the background colour, a text field sets the greeting, and a button changes colour when you click it.

Conclusion

DOM elements provide a powerful way to make your p5.js projects more interactive and user-friendly. Whether you want simple buttons or complex forms, the p5.js DOM functions make it easy to create and manage these elements.
