An L-system is a rule system that rewrites text. By reading that text as a drawing afterwards, you get growth forms like plants, branches, and fractals.

What you will learn
  • Thinking in rules and rewriting starting from an axiom
  • Describing growth with a grammar instead of drawing directly
  • Reading a string with turtle commands: F, +, -, [, ]
  • Creating branches with push() and pop()
  • Seeing the connection between text structure and image structure

Why this concept is fascinating

With an L-system, you don't draw a plant step by step by hand. You only describe how it grows. The image emerges from a grammar.

Think about it: can an image have a kind of language?

How does it work?

You start with an axiom, a starting string, and repeatedly replace each letter using rules.

Axiom:     F
Rule:      F -> F+F-F-F+F

Iteration 0: F
Iteration 1: F+F-F-F+F
Iteration 2: F+F-F-F+F + F+F-F-F+F - ...

Then you read the string with turtle commands:

  • F - move forward and draw
  • + - turn right
  • - - turn left
  • [ - save position and direction
  • ] - restore position and direction

Plant model

With [ and ] you can create branches:

Axiom: X
Rules: X -> F+[[X]-X]-F[-FX]+X
       F -> FF
Angle: 25
for (let c of sentence) {
  if (c === "F") { line(0, 0, len, 0); translate(len, 0); }
  else if (c === "+") rotate(radians(angle));
  else if (c === "-") rotate(radians(-angle));
  else if (c === "[") push();
  else if (c === "]") pop();
}

This way you get something that feels like growth: each iteration builds on the previous structure.

What does a rule change?

A small change in a rule or angle can make a plant elegant, compact, chaotic, or almost architectural.

  • more iterations - more detail
  • larger angle - wider branching
  • different replacement rule - different growth behavior

Exercise

Change only one thing at a time: first the angle, then the number of iterations, then the rule for X. Each time, describe whether your result looks more like a plant, a network, an ornament, or an abstract pattern.
