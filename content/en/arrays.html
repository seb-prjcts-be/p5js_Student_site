Arrays are lists of values. They are perfect for storing multiple related elements, like positions of balls, colors, or objects in an animation.

What you will learn
  • Creating arrays with [] and accessing them via index (starts at 0)
  • Adding elements with push()
  • Looping through arrays with for and the modern for...of
  • Object arrays as the basis of a simple particle system
  • Working functionally with filter(), map() and forEach()

Creating Arrays

You create an array like this:

let numbers = [10, 20, 30, 40];
let colors = ["red", "green", "blue"];
let empty = [];  // Empty array

How does an index work?

Every element has an index: its position number in the array. The first slot is 0, not 1.

let colors = ["red", "green", "blue", "yellow"];
//            ↑      ↑        ↑       ↑
// index:     0      1        2       3

console.log(colors[0]);  // "red"    (first)
console.log(colors[2]);  // "blue"
console.log(colors[3]);  // "yellow" (last)
console.log(colors[4]);  // undefined — doesn't exist

Array length

.length returns the number of elements. So the last element sits at index length - 1:

let numbers = [10, 20, 30, 40];
console.log(numbers.length);                    // 4
console.log(numbers[numbers.length - 1]);       // 40 — last element

Adding Elements

Use push() to add elements to the end:

let balls = [];

function mousePressed() {
  balls.push(mouseX);  // Add mouse position
}

console.log(balls);  // [120, 340, 55, ...]

Looping Through Arrays

In a for loop i counts from 0 up to positions.length - 1. At each step positions[i] grabs the matching element:

let positions = [50, 150, 250, 350, 450];

function draw() {
  background(245);
  for (let i = 0; i < positions.length; i++) {
    circle(positions[i], height / 2, 40);
  }
}

What happens at each step?

// i = 0 → positions[0] = 50   → circle(50,  height/2, 40)
// i = 1 → positions[1] = 150  → circle(150, height/2, 40)
// i = 2 → positions[2] = 250  → circle(250, height/2, 40)
// i = 3 → positions[3] = 350  → circle(350, height/2, 40)
// i = 4 → positions[4] = 450  → circle(450, height/2, 40)

Or the modern for...of loop - shorter and more readable:

for (let x of positions) {
  circle(x, height / 2, 40);
}

Object Arrays

Arrays can also contain objects. This lets you store multiple properties per item:

let balls = [
  { x: 100, y: 200, size: 40, color: [255, 0, 0] },
  { x: 300, y: 150, size: 60, color: [0, 0, 255] },
  { x: 450, y: 300, size: 30, color: [0, 255, 0] }
];

function draw() {
  background(30);
  for (let ball of balls) {
    fill(ball.color);
    circle(ball.x, ball.y, ball.size);
  }
}

This is the basis of a particle system: an array of objects that each track their own properties.
