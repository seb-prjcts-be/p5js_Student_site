Boids zijn simpele agents die samen zwermgedrag vormen. Elke boid volgt een paar regels, maar samen lijken ze bijna intelligent te bewegen.

Waarom dit concept boeiend is

Boids tonen dat een groep geen leider nodig heeft om toch gecoordineerd te lijken. De orde zit niet in een centraal plan, maar in de interactie tussen buren.

Denkvraag: heb je een leider nodig om collectief gedrag te krijgen?

Wat je hier leert

  • werken met agents die elk hun eigen toestand bijhouden
  • vectors gebruiken voor richting en snelheid
  • zien hoe lokale interactie een groepspatroon maakt
  • balanceren tussen aantrekken, afstoten en uitlijnen

De drie klassieke regels

  • alignment - beweeg ongeveer in dezelfde richting als je buren
  • cohesion - blijf in de buurt van de groep
  • separation - bots niet op elkaar

Die drie regels samen zorgen voor flocking: gedrag dat lijkt op vogels, vissen of een zwerm drones.

In code

Elke boid berekent krachten op basis van nabije boids:

for (let andere of boids) {
  let d = dist(this.x, this.y, andere.x, andere.y);

  if (d < zichtRadius) {
    alignment.add(andere.vel);
    cohesion.add(andere.pos);
  }

  if (d > 0 && d < botsRadius) {
    separation.add(verschilVector);
  }
}

Daarna worden die krachten beperkt en opgeteld bij de snelheid.

Oefening

Bouw eerst alleen separation. Voeg daarna alignment toe, en pas daarna cohesion. Zo voel je echt welke regel welk effect heeft. Welke combinatie oogt rustig, welke zenuwachtig?
