Arrays zijn lijsten van waarden. Ze zijn perfect voor het opslaan van meerdere gerelateerde elementen, zoals posities van ballen, kleuren of objecten in een animatie.

Kernpunten
  • Arrays aanmaken met [] en benaderen via index (start op 0)
  • Elementen toevoegen met push()
  • Arrays doorlopen met for en het modernere for...of
  • Object-arrays als basis voor een eenvoudig particle system
  • Functioneel werken met filter(), map() en forEach()

Arrays Aanmaken

Een array maak je zo aan:

let getallen = [10, 20, 30, 40];
let kleuren = ["rood", "groen", "blauw"];
let leeg = [];  // Lege array

Hoe werkt een index?

Elk element heeft een index: zijn plaatsnummer in de array. De eerste plek is 0, niet 1.

let kleuren = ["rood", "groen", "blauw", "geel"];
//            ↑       ↑        ↑        ↑
// index:     0       1        2        3

console.log(kleuren[0]);  // "rood"   (eerste)
console.log(kleuren[2]);  // "blauw"
console.log(kleuren[3]);  // "geel"   (laatste)
console.log(kleuren[4]);  // undefined — bestaat niet

Array lengte

.length geeft het aantal elementen terug. Het laatste element vind je dus op index length - 1:

let getallen = [10, 20, 30, 40];
console.log(getallen.length);                   // 4
console.log(getallen[getallen.length - 1]);     // 40 — laatste element

Elementen Toevoegen

Gebruik push() om elementen toe te voegen aan het einde:

let ballen = [];

function mousePressed() {
  ballen.push(mouseX);  // Voeg muispositie toe
}

console.log(ballen);  // [120, 340, 55, ...]

Arrays Doorlopen met een Loop

Met een for-loop telt i van 0 tot posities.length - 1. Bij elke stap haalt posities[i] het bijhorende element op:

let posities = [50, 150, 250, 350, 450];

function draw() {
  background(245);
  for (let i = 0; i < posities.length; i++) {
    circle(posities[i], height / 2, 40);
  }
}

Wat gebeurt er bij elke stap?

// i = 0 → posities[0] = 50   → circle(50,  height/2, 40)
// i = 1 → posities[1] = 150  → circle(150, height/2, 40)
// i = 2 → posities[2] = 250  → circle(250, height/2, 40)
// i = 3 → posities[3] = 350  → circle(350, height/2, 40)
// i = 4 → posities[4] = 450  → circle(450, height/2, 40)

Of de moderne for...of loop - korter en leesbaarder:

for (let x of posities) {
  circle(x, height / 2, 40);
}

Object Arrays

Arrays kunnen ook objecten bevatten. Zo sla je meerdere eigenschappen per item op:

let ballen = [
  { x: 100, y: 200, grootte: 40, kleur: [255, 0, 0] },
  { x: 300, y: 150, grootte: 60, kleur: [0, 0, 255] },
  { x: 450, y: 300, grootte: 30, kleur: [0, 255, 0] }
];

function draw() {
  background(30);
  for (let bal of ballen) {
    fill(bal.kleur);
    circle(bal.x, bal.y, bal.grootte);
  }
}

Dit is de basis van een particle system: een array van objecten die elk hun eigen eigenschappen bijhouden.

In het hoofdstuk Objecten & classes zie je hoe je een array van class-instanties bouwt. Dat is de volgende stap: elk item in de array krijgt dan eigen methoden.
