Arrays zijn lijsten van waarden. Ze zijn perfect voor het opslaan van meerdere gerelateerde elementen, zoals posities van ballen, kleuren of objecten in een animatie.

Arrays Aanmaken

Een array maak je zo aan:

let getallen = [10, 20, 30, 40];
let namen = ["rood", "groen", "blauw"];
let leeg = [];  // Lege array

// Toegang via index (begint bij 0)
console.log(getallen[0]);  // 10
console.log(namen[2]);     // "blauw"

Array lengte

let getallen = [10, 20, 30, 40];
console.log(getallen.length);  // 4

Elementen Toevoegen en Verwijderen

Gebruik push() om elementen toe te voegen aan het einde:

let ballen = [];

function mousePressed() {
  ballen.push(mouseX);  // Voeg muispositie toe
}

console.log(ballen);  // [120, 340, 55, ...]

Andere handige methoden:

  • push(waarde) — voeg toe aan einde
  • pop() — verwijder laatste element (en geef het terug)
  • splice(index, aantal) — verwijder elementen op een positie
  • indexOf(waarde) — zoek de index van een waarde

Arrays Doorlopen met een Loop

Gebruik een for-loop om elk element te verwerken:

let posities = [50, 150, 250, 350, 450];

function draw() {
  background(245);
  for (let i = 0; i < posities.length; i++) {
    circle(posities[i], height / 2, 40);
  }
}

Of de moderne for...of loop — korter en leesbaarder:

for (let x of posities) {
  circle(x, height / 2, 40);
}

Object Arrays

Arrays kunnen ook objecten bevatten. Zo sla je meerdere eigenschappen per item op:

let ballen = [
  { x: 100, y: 200, grootte: 40, kleur: [255, 0, 0] },
  { x: 300, y: 150, grootte: 60, kleur: [0, 0, 255] },
  { x: 450, y: 300, grootte: 30, kleur: [0, 255, 0] }
];

function draw() {
  background(30);
  for (let bal of ballen) {
    fill(bal.kleur);
    circle(bal.x, bal.y, bal.grootte);
  }
}

Dit is de basis van een particle system: een array van objecten die elk hun eigen eigenschappen bijhouden.

Dynamische Object Arrays

Je kunt objecten aanmaken en toevoegen terwijl je sketch draait:

let ballen = [];

function mousePressed() {
  // Voeg een nieuw object toe bij klik
  ballen.push({
    x: mouseX,
    y: mouseY,
    vx: random(-2, 2),   // Horizontale snelheid
    vy: random(-4, -1),  // Verticale snelheid (omhoog)
    grootte: random(15, 40)
  });
}

function draw() {
  background(245);

  for (let bal of ballen) {
    bal.x += bal.vx;
    bal.y += bal.vy;
    bal.vy += 0.15;  // Zwaartekracht

    circle(bal.x, bal.y, bal.grootte);
  }

  // Verwijder ballen die buiten het scherm zijn
  ballen = ballen.filter(bal => bal.y < height + 50);
}

Handige Array Methoden

filter() — elementen selecteren

// Verwijder dode particles (life <= 0)
particles = particles.filter(p => p.life > 0);

map() — elk element omzetten

let getallen = [1, 2, 3, 4, 5];
let verdubbeld = getallen.map(n => n * 2);
// verdubbeld = [2, 4, 6, 8, 10]

forEach() — iets doen met elk element

ballen.forEach(bal => {
  bal.x += bal.vx;
  bal.y += bal.vy;
});
