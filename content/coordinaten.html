Elke vorm die je in p5.js tekent heeft coördinaten nodig. Het canvas heeft een eigen assenstelsel dat standaard een beetje afwijkt van wat je uit de wiskundeles kent.

Kernpunten
  • Het p5.js-assenstelsel: (0, 0) ligt linksboven
  • X loopt naar rechts, Y loopt naar beneden - anders dan in de wiskunde
  • De canvasgrootte bepaalt de grenzen van je coördinaten
  • Elk punt op het canvas heeft een (x, y)-coördinaat
  • Experimenteren met coördinaten via een live editor

Assenstelsel

Een canvas heeft een coördinatenstelsel waarmee je elk punt op het canvas kunt aanwijzen. De oorsprong (0, 0) ligt linksboven. X loopt naar rechts, Y loopt naar beneden.

Weetje

Dit is anders dan wiskunde waar Y omhoog gaat! Veel beeldbewerking gebruikt de linksboven-oorsprong omdat ook de pixels van een scherm van boven naar beneden worden geadresseerd.

Van coördinaat naar vorm

Elk punt op het canvas heeft een x- en y-waarde. Als je een cirkel op (200, 200) tekent, staat het middelpunt van die cirkel precies op de plek waar x=200 en y=200 is.

// Punt precies in het midden van een 400×400 canvas:
point(200, 200);

// Cirkel linksboven:
circle(50, 50, 40);

// Cirkel rechtsonder:
circle(350, 350, 40);

Vijf cirkels op herkenbare plekken. Pas de getallen aan en zie waar de cirkels verschijnen.

Canvasgrootte en coördinaten

De canvasgrootte bepaalt welke coördinaten je kunt gebruiken. Op een canvas van 400 × 300 loopt x van 0 tot 400 en y van 0 tot 300. Een punt op (500, 100) valt buiten het canvas en is dus niet zichtbaar.

Gebruik width en height in plaats van de getallen zelf. Zo blijft je code kloppen als je de canvasgrootte verandert.

function setup() {
  createCanvas(400, 300);

  // Middelpunt, ongeacht de canvasgrootte:
  circle(width / 2, height / 2, 60);
}
