Wat als je sketch kan zien, horen of reageren op beweging - zonder dat je zelf een AI-model hoeft te bouwen? Met ml5.js kun je kant-en-klare AI-modellen inladen in p5.js en meteen creatief aan de slag.

Wat is ml5.js?

ml5.js is een JavaScript-bibliotheek die machine learning toegankelijk maakt voor makers en ontwerpers. Het is speciaal ontworpen om samen te werken met p5.js.

Het idee: jij hoeft geen wiskundige te zijn om AI te gebruiken in je creatieve werk. ml5.js biedt voorgetrainde modellen die je direct kunt aanspreken - voor lichaamsherkenning, geluidsclassificatie, beeldherkenning en meer.

  • Gebouwd bovenop TensorFlow.js, maar veel eenvoudiger in gebruik
  • Werkt direct in de browser - geen installatie nodig
  • Open source en gratis

ml5.js inladen

Om ml5.js te gebruiken in je project, voeg je het toe via een CDN-link. In de live editor op deze site kun je het direct gebruiken in een <script>-tag:

<script src="https://unpkg.com/ml5@1/dist/ml5.min.js"></script>

In een eigen HTML-bestand plaats je deze regel in de <head>, naast je p5.js-script.

Beschikbare modellen

ml5.js biedt verschillende voorgetrainde modellen. Hier zijn de belangrijkste:

BodyPose

Herkent de positie van het menselijk lichaam via de webcam. Detecteert punten zoals schouders, ellebogen, polsen, knieën en enkels. Ideaal voor interactieve installaties of bewegingsgestuurde sketches.

HandPose

Herkent de positie van handen en individuele vingers. Kan 21 punten per hand detecteren. Handig voor gebarenherkenning of het tekenen met je handen.

FaceMesh

Detecteert 468 punten op het gezicht. Hiermee kun je gezichtsfilters maken, emoties analyseren of gezichtsbewegingen omzetten naar visuele effecten.

ImageClassifier

Herkent objecten in afbeeldingen. Geeft een label terug ("kat", "auto", "bloem") met een betrouwbaarheidsscore. Werkt met foto's en webcambeelden.

SoundClassifier

Herkent geluiden via de microfoon. Kan onderscheid maken tussen spreken, klappen, fluiten en achtergrondgeluid.

Je eerste ml5.js-sketch: BodyPose

Laten we stap voor stap een sketch bouwen die je lichaam detecteert via de webcam en de herkenningspunten tekent.

Stap 1: webcam activeren

let video;

function setup() {
  createCanvas(640, 480);
  video = createCapture(VIDEO);
  video.size(640, 480);
  video.hide(); // Verberg het HTML-video-element
}

function draw() {
  image(video, 0, 0); // Teken de video op het canvas
}

Stap 2: BodyPose laden

In moderne p5.js gebruik je async function setup() met await, zodat het model klaar is voor je verder gaat:

let video;
let bodyPose;
let poses = [];

async function setup() {
  createCanvas(640, 480);
  video = createCapture(VIDEO);
  video.size(640, 480);
  video.hide();

  // Wacht tot het BodyPose-model geladen is
  bodyPose = await ml5.bodyPose();

  // Start met het detecteren van poses
  bodyPose.detectStart(video, ontvangResultaten);
}

function ontvangResultaten(resultaten) {
  poses = resultaten;
}

Stap 3: punten tekenen

function draw() {
  image(video, 0, 0);

  // Teken alle gedetecteerde punten
  for (let i = 0; i < poses.length; i++) {
    let pose = poses[i];
    for (let j = 0; j < pose.keypoints.length; j++) {
      let punt = pose.keypoints[j];
      // Teken alleen als de betrouwbaarheid hoog genoeg is
      if (punt.confidence > 0.2) {
        fill(0, 255, 0);
        noStroke();
        circle(punt.x, punt.y, 10);
      }
    }
  }
}

Elke keypoint heeft een x, y en confidence waarde. De confidence geeft aan hoe zeker het model is van de detectie (0 tot 1).

Van detectie naar creatie

Het wordt pas echt interessant als je de gedetecteerde punten gebruikt om iets visueel te doen. Enkele ideeën:

  • Lijnen tussen punten - Verbind de keypoints met lijnen om een skelet te tekenen
  • Deeltjes volgen - Laat deeltjes zwermen rond je handen of hoofd
  • Kleur met je lichaam - Gebruik de positie van je handen om kleuren te mengen
  • Geluid triggeren - Koppel beweging aan geluidsparameters
  • Abstracte vormen - Vervang het videobeeld door abstracte geometrie die je lichaam volgt

Belangrijk om te weten

  • Webcam vereist - De meeste ml5.js-modellen gebruiken de webcam. Zorg dat je browser toestemming heeft.
  • Eerste keer laden duurt even - Het model wordt gedownload bij het eerste gebruik. Daarna gaat het sneller.
  • Privacy - Alle verwerking gebeurt lokaal in je browser. Er worden geen beelden naar een server gestuurd.
  • Prestaties - AI-modellen in de browser zijn zwaarder dan gewone sketches. Op oudere computers kan het trager lopen.
