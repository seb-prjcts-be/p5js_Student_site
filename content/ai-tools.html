AI-tools zijn er in alle soorten en maten. Van playgrounds om modellen te testen tot editors die mee programmeren - dit is een overzicht van de meest nuttige tools voor studenten en makers.

Wat is een API?

Een API (Application Programming Interface) is een manier om twee softwaresystemen met elkaar te laten praten. Je stuurt een vraag, en je krijgt een antwoord terug - zonder te weten hoe het aan de andere kant werkt.

Vergelijk het met een restaurant: jij bestelt (de request), de keuken bereidt het (de verwerking), en de ober brengt het resultaat (de response). Jij hoeft niet te weten hoe de keuken werkt.

Bijna alle AI-tools werken via een API. Als je code schrijft die een AI-model aanspreekt, stuur je een verzoek naar zo'n API en verwerk je het antwoord in je eigen applicatie.

Geheugen en context

AI-modellen starten elk gesprek blanco - ze herinneren zich niets van vorige sessies. Dat klinkt als een beperking, maar er zijn slimme manieren om daar omheen te werken:

Geef context mee

Je kunt een AI-model altijd extra informatie meegeven in je prompt. Plak er een stuk tekst bij, geef een link, of beschrijf je project. Hoe meer relevante context, hoe beter het antwoord.

CLAUDE.md — geheugen voor je project

Tools zoals Claude Code gebruiken een CLAUDE.md-bestand als langetermijngeheugen. Daarin beschrijf je je project, je voorkeuren en regels. Bij elke sessie leest de AI dat bestand en weet meteen waar je mee bezig bent.

Je kunt dit principe overal toepassen: maak een tekstbestand met context over je project en plak dat aan het begin van elk nieuw gesprek.

Skills en instructies

Veel AI-tools ondersteunen custom instructions of skills - herbruikbare instructies die je opslaat. Zo hoef je niet elke keer opnieuw uit te leggen wat je wilt. Denk aan het als een recept dat de AI telkens opnieuw kan volgen.

Kennisgrens

Modellen hebben een kennisgrens (cutoff date): recente gebeurtenissen kennen ze niet automatisch. Maar veel tools kunnen tegenwoordig het web doorzoeken, en je kunt altijd zelf actuele info meegeven in je prompt.

AI Labs en Playgrounds

De grote AI-bedrijven bieden eigen playgrounds aan: omgevingen waar je modellen direct kunt testen zonder te programmeren.

Google AI Studio

Google AI Studio is Google's gratis playground voor Gemini-modellen. Je kunt er tekst, afbeeldingen en code testen, en je eigen API-sleutel aanmaken.

  • Probeer multimodale prompts: geef een afbeelding mee met je vraag
  • Bekijk de tokenizer: zie hoe je tekst wordt opgedeeld
  • Gratis te gebruiken voor experimenteren

OpenAI Platform

OpenAI Platform is de officiële speeltuin van OpenAI (makers van ChatGPT). Je kunt hier GPT-modellen testen, fine-tuning verkennen en API-aanvragen analyseren.

Anthropic Console

Anthropic Console is de playground van Anthropic (makers van Claude). Vergelijkbaar met de anderen - handig om Claude's sterktes te verkennen, zoals langere documenten verwerken en nauwkeurig redeneren.

NotebookLM

NotebookLM (van Google) is een tool die AI combineert met je eigen bronnen. Je laadt documenten, PDF's of URLs in, en daarna kun je er vragen over stellen - het model antwoordt alleen op basis van wat je hebt toegevoegd.

Heel nuttig voor:

  • Een lange PDF samenvatten en bevragen
  • Notities uit verschillende bronnen samenvoegen
  • Een "audio overview" genereren: een gesimuleerd podcastgesprek over je bronnen

NotebookLM hallucineert veel minder dan gewone chatbots, omdat het vastzit aan de documenten die je hebt opgegeven.

Chrome-browser tips

Je browser is zelf een krachtig AI-startpunt geworden:

  • Gemini in Chrome - rechtstreeks in de adresbalk kun je vragen stellen door @gemini te typen
  • Extensies - tools zoals Monica, Merlin of Sider voegen een AI-zijpaneel toe aan elke pagina
  • Lees- en samenvattingstools - selecteer tekst op een pagina en vraag direct om een samenvatting

Andere nuttige AI-tools

Gamma

Gamma maakt presentaties, documenten en websites op basis van een tekstprompt. Handig als startpunt - pas daarna aan naar eigen stijl.

ElevenLabs

ElevenLabs biedt tekst-naar-spraak van hoge kwaliteit. Genereer stemmen in tientallen talen en stijlen. Gebruik het voor video's, prototypes of experimenten met audio.

Napkin

Napkin zet tekst automatisch om in diagrammen en infographics. Handig voor het visualiseren van processen of ideeën.

Eightify

Eightify is een Chrome-extensie die YouTube-video's samenvat. Krijg de kernpunten van een lange video in seconden.

Programmeren met AI

AI verandert ook hoe we code schrijven. Er zijn tools die rechtstreeks in je code-editor zitten en mee suggereren, uitleggen en aanvullen.

GitHub Copilot (in VSCode)

Copilot suggereert code terwijl je typt - op basis van de context van je bestand. Beschikbaar als extensie in VSCode. Gratis voor studenten via het GitHub Student Developer Pack.

Cursor

Cursor is een aparte code-editor gebouwd rondom AI. Je kunt er volledige codeblokken laten schrijven, uitleggen of refactoren via gewone taal. Werkt goed voor grotere projecten.

Tips bij AI-geassisteerd programmeren

  • Beschrijf wat je wilt bereiken, niet hoe - laat de AI de implementatie kiezen
  • Controleer altijd de gegenereerde code - AI maakt fouten
  • Gebruik AI voor uitleg: "Leg uit wat deze functie doet" werkt uitstekend
  • Kleine stappen: laat AI één functie tegelijk schrijven, niet een heel project
