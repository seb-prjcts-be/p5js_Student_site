Sonificatie is het proces van het omzetten van data in geluid. Waar we gewone data meestal visualiseren met grafieken en kleur, laat sonificatie je de data horen. Dat voegt een extra dimensie toe aan je p5.js-projecten, zeker als je werkt met externe data zoals CSV-bestanden, JSON of via API's.

Kernpunten
  • Data uit een bestand of API mappen op eigenschappen van geluid (toonhoogte, duur, volume)
  • Werken met de p5.Sound-library voor oscillators en samples
  • Data beluisteren als een tweede laag naast visualisatie
  • Voorbeeld: NASA-meteorietendata beluisteren en tegelijk op een kaart tonen
  • Kritische reflectie: wanneer voegt geluid echt iets toe en wanneer stoort het?

Wat is sonificatie?

Sonificatie neemt getallen uit je data en vertaalt ze naar hoorbare eigenschappen. Een rij temperaturen kan een melodielijn worden waarin warmer = hoger, een dataset van aardbevingen kan pulsen met intensiteit geven, en een lange reeks verkoopscijfers kan een akkoord vormen dat zacht of fel klinkt.

Het voordeel? Ons gehoor pikt patronen op die we met het oog missen - ritme, herhaling, plotse pieken. Geluid speelt zich af in de tijd en sleept de luisteraar vanzelf door de data.

De p5.Sound-library

Om met geluid te werken in p5.js heb je de p5.Sound-bibliotheek nodig. Die biedt klassen voor oscillators, samples, filters en analyse.

p5.Sound moet apart ingeladen worden naast p5.js zelf:

<script src="https://cdn.jsdelivr.net/npm/p5@2.2.3/lib/p5.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/p5@2.2.3/lib/addons/p5.sound.min.js"></script>
Let op

Browsers laten geluid pas starten na een gebruikersactie (een klik of toetsaanslag). Zet je geluid dus bij voorkeur aan in mousePressed() of met userStartAudio().

Een data-waarde hoorbaar maken

De kleinste bouwsteen: een oscillator met een frequentie die je uit je data haalt.

let osc;

function setup() {
  createCanvas(400, 200);
  osc = new p5.Oscillator('sine');
  osc.amp(0);
  osc.start();
}

function draw() {
  background(245);

  // mouseX (0..width) mappen op 200 Hz..800 Hz
  let freq = map(mouseX, 0, width, 200, 800);
  osc.freq(freq);

  text("Frequentie: " + round(freq) + " Hz", 20, 30);
}

function mousePressed() {
  osc.amp(0.3, 0.1);  // Fade in
}

function mouseReleased() {
  osc.amp(0, 0.5);    // Fade out
}

Dezelfde logica werkt voor elke datastroom: een waarde uit een CSV-kolom, een JSON-veld of een API-response mappen op frequentie, volume of duur.

Voorbeeld: meteorieten beluisteren

Seb maakte ooit - lang geleden, toen stemsoftware nog kraste - een sonificatie van meteorieteninslagen uit de NASA Meteorite Landings API. Elke landing werd een plekje op een wereldkaart, de locatienaam werd uitgesproken door spraaksynthese, en de massa bepaalde de toonhoogte. Stilaan verscheen de wereldkaart uit de data zelf.

Weetje

Sonificatie wordt ook serieus ingezet: NASA stuurt ruimte- en telescoopdata door sonificatie-software zodat blinde wetenschappers zwarte gaten kunnen "horen". Zoek eens naar "NASA sonification black hole".

Mapping-strategieën

De kunst van sonificatie zit in de mapping: welke data-eigenschap stuurt welke geluidseigenschap?

  • Waarde → toonhoogte - intuïtief: groot = hoog, klein = laag
  • Frequentie → ritme - veel datapunten dichtbij elkaar worden snelle pulsen
  • Categorie → timbre - verschillende data-soorten krijgen elk een eigen instrument
  • Positie → stereo-panning - x-as bepaalt links/rechts in het geluidsbeeld
  • Tijd in data → tijd in geluid - een jaar = een seconde, of desnoods een milliseconde

Vermijd mapping die niets toevoegt. Als je data al rustig zijn, hoeft het geluid niet constant te piepen. Soms is stilte de beste sonificatie van "er is niets gebeurd".

Oefening

Kies een kleine dataset (bv. temperatuur per uur, een rij scores, wisselkoersen van een week). Kies één getal per datapunt en map dat op de frequentie van een oscillator. Speel de reeks af over vijf seconden. Beschrijf wat je hoort dat je op een grafiek niet meteen zou zien.

Conclusie

Sonificatie is een unieke manier om data te verkennen en te begrijpen. Het voegt een extra laag interactiviteit en beleving toe aan je p5.js-projecten. Door geluid te combineren met visuele data bouw je rijke, meeslepende, actuele ervaringen. Kortom: leer je oren kijken.
