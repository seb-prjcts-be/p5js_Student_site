Loops herhalen code. Ze zijn perfect voor het tekenen van meerdere elementen of het doorlopen van data.

For-Loop

De for-loop is het meest gebruikt. Hij heeft drie delen:

for (start; voorwaarde; stap) {
  // Code die herhaald wordt
}

Voorbeeld: 10 Cirkels

for (let i = 0; i < 10; i++) {
  ellipse(i * 50, 100, 40, 40);
}

Dit tekent 10 cirkels naast elkaar.

Loop Variabelen

De loop variabele (meestal i) verandert bij elke iteratie:

for (let i = 0; i < 5; i++) {
  console.log(i);  // Print: 0, 1, 2, 3, 4
}

Gebruik in Tekenen

for (let i = 0; i < 10; i++) {
  let x = i * 50;
  let y = 100;
  ellipse(x, y, 30, 30);
}

Geneste loops

Je kunt loops in loops gebruiken voor 2D patronen - een loop voor de rijen (y), een loop voor de kolommen (x):

for (let x = 0; x < 10; x++) {
  for (let y = 0; y < 10; y++) {
    ellipse(x * 50, y * 50, 30, 30);
  }
}

Dit tekent een grid van 10×10 cirkels.

Voorbeeld met celgrootte

Een handige techniek is werken met een variabele celgrootte in plaats van vaste afstanden:

let celgrootte = 40;
let grootte = 400;

for (let j = 0; j < grootte; j += celgrootte) {
  for (let i = 0; i < grootte; i += celgrootte) {
    rect(i, j, celgrootte);  // Vierkantje op positie (i, j)
  }
}

De i staat voor alle x-posities, de j voor alle y-posities - zo is de afspraak onder programmeurs.

Kunstenares Vera Molnár gebruikte precies dit principe (raster van cellen) om haar tekenrobot aan te sturen. Bekijk de Vera Molnár pagina voor voorbeelden.

Voorbeeld: Desordres-stijl

Molnárs beroemde werk (Des)Ordres (1974) gebruikt een geneste loop om concentrische vierkanten te tekenen, met een kleine willekeurige afwijking per hoekpunt. Het resultaat: orde met net genoeg chaos.

let celgrootte = 80;

function setup() {
  createCanvas(400, 400);
  noFill();
  stroke(0);
  noLoop();
}

function draw() {
  background(255);
  for (let y = 0; y < height; y += celgrootte) {
    for (let x = 0; x < width; x += celgrootte) {
      // 4 concentrische vierkanten per cel
      for (let k = 0; k < 4; k++) {
        let afwijking = k * 1.5;  // Meer afwijking bij binnenste vierkant
        let marge = k * 10;
        beginShape();
        vertex(x + marge + random(-afwijking, afwijking),
               y + marge + random(-afwijking, afwijking));
        vertex(x + celgrootte - marge + random(-afwijking, afwijking),
               y + marge + random(-afwijking, afwijking));
        vertex(x + celgrootte - marge + random(-afwijking, afwijking),
               y + celgrootte - marge + random(-afwijking, afwijking));
        vertex(x + marge + random(-afwijking, afwijking),
               y + celgrootte - marge + random(-afwijking, afwijking));
        endShape(CLOSE);
      }
    }
  }
}

Drie geneste loops: rijen, kolommen en de concentrische vierkanten per cel. De random()-afwijking maakt elk vierkant net iets anders.

While-Loop

Een while-loop herhaalt zolang een voorwaarde waar is:

let x = 0;
while (x < width) {
  line(x, 0, x, height);
  x += 20;
}

Let op: zorg dat de voorwaarde ooit false wordt, anders krijg je een oneindige loop!
