Loops herhalen code. Ze zijn perfect voor het tekenen van meerdere elementen of het doorlopen van data.

For-Loop

De for-loop is het meest gebruikt. Hij heeft drie delen:

for (start; voorwaarde; stap) {
  // Code die herhaald wordt
}

Voorbeeld: 10 Cirkels

for (let i = 0; i < 10; i++) {
  ellipse(i * 50, 100, 40, 40);
}

Dit tekent 10 cirkels naast elkaar.

Loop Variabelen

De loop variabele (meestal i) verandert bij elke iteratie:

for (let i = 0; i < 5; i++) {
  console.log(i);  // Print: 0, 1, 2, 3, 4
}

Gebruik in Tekenen

for (let i = 0; i < 10; i++) {
  let x = i * 50;
  let y = 100;
  ellipse(x, y, 30, 30);
}

Geneste Loops

Je kunt loops binnen loops gebruiken voor 2D patronen:

for (let x = 0; x < 10; x++) {
  for (let y = 0; y < 10; y++) {
    ellipse(x * 50, y * 50, 30, 30);
  }
}

Dit tekent een grid van 10x10 cirkels.

While-Loop

Een while-loop herhaalt zolang een voorwaarde waar is:

let x = 0;
while (x < width) {
  line(x, 0, x, height);
  x += 20;
}

Let op: zorg dat de voorwaarde ooit false wordt, anders krijg je een oneindige loop!
