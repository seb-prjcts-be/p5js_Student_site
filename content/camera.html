Met één functie kan p5.js jouw webcam aanspreken: createCapture(). Dat opent de deur naar real-time interactie, live effecten en kunstwerken die reageren op wat er voor het scherm gebeurt.

Kernpunten
  • Webcam-toegang via createCapture(VIDEO)
  • Het video-element verbergen met v.hide() zodat enkel het canvas zichtbaar blijft
  • Filters toepassen: GRAY, INVERT, POSTERIZE, BLUR
  • Webcam combineren met andere p5.js-functies voor interactieve sketches
Let op

Let er goed op dat de camera van je laptop werkt en dat er geen beschermer voor geschoven zit. Test je camera door "Camera" te typen in het zoekveld van je Windows-computer. De browser vraagt de eerste keer om toestemming om de webcam te gebruiken.

Een eenvoudige webcam-sketch

Met createCapture(VIDEO) krijg je een video-object dat continu nieuwe beelden van je webcam levert. Je tekent het op het canvas met image(), net zoals een gewone afbeelding. Het losse HTML-videoelement dat p5 aanmaakt verberg je met v.hide(), anders zie je de feed twee keer.

let v;

function setup() {
  createCanvas(400, 400);
  v = createCapture(VIDEO);
  v.hide(); // anders krijg je twee keer hetzelfde videobeeld
}

function draw() {
  image(v, 0, 0);
}

Interactie met de webcam

Je kunt de webcam-data combineren met andere p5.js-functies om interactieve toepassingen te maken. Bijvoorbeeld: een object tekenen dat reageert op beweging in het beeld, of een gek noise-effect toevoegen.

Effecten en filters

Filters kan je toepassen op afbeeldingen, video's en eigenlijk op je hele canvas. Of je nu een vintage look wil of real-time effecten op een webcam-feed: met één regel kan je veel bereiken.

p5.js biedt een aantal kant-en-klare filters via de functie filter(). Veel gebruikte constantes zijn GRAY, INVERT, POSTERIZE en BLUR.

Filter op de webcam

let v;

function setup() {
  createCanvas(400, 400);
  v = createCapture(VIDEO);
  v.hide();
}

function draw() {
  image(v, 0, 0);
  filter(INVERT);
}

Probeer POSTERIZE(3) of BLUR(5) met een parameter, of stapel meerdere filters boven elkaar voor onverwachte combinaties.
