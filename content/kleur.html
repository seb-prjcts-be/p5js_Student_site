Kleur maakt je sketches levend. p5.js biedt verschillende manieren om kleuren te gebruiken.

Kleur functies

Je stelt kleur altijd in vóór je de vorm tekent. p5.js leest de code van boven naar onder:

  • background(kleur) - achtergrondkleur van het canvas
  • fill(kleur) - vulkleur voor vormen
  • stroke(kleur) - lijnkleur (omtreklijn)
  • strokeWeight(dikte) - dikte van de omtreklijn in pixels
  • noFill() - geen vulling (transparant binnenste)
  • noStroke() - geen omtreklijn
stroke(255, 0, 0);       // Rode omtreklijn
strokeWeight(3);          // Lijndikte 3 pixels
fill(0, 200, 255);        // Lichtblauwe vulling
circle(200, 200, 100);

noStroke();               // Geen omtreklijn voor volgende vormen
fill(255, 150, 0);
rect(100, 50, 80, 80);

Kleurwaarden

Kleuren kunnen op verschillende manieren worden opgegeven:

Grijs (0-255)

fill(255);        // Wit
fill(128);        // Grijs
fill(0);          // Zwart

RGB (Rood, Groen, Blauw)

fill(255, 0, 0);      // Rood
fill(0, 255, 0);      // Groen
fill(0, 0, 255);      // Blauw
fill(255, 255, 0);    // Geel

RGBA (Met Transparantie)

fill(255, 0, 0, 128);  // Half transparant rood
fill(0, 0, 255, 200);  // Bijna ondoorzichtig blauw

Kleurnamen

Je kunt ook CSS-kleurnamen gebruiken als tekst:

fill('red');
rect(100, 100, 200, 100);

fill('white');
square(100, 200, 100);

Andere kleurnotaties

Naast RGB kun je kleuren ook opgeven als hexadecimale code (zoals in CSS) of als CSS-naam:

// Hexadecimale code (# + 6 cijfers)
fill('#FF6600');    // Oranje
stroke('#003399'); // Donkerblauw

// CSS-kleurnamen
fill('tomato');
fill('steelblue');
fill('gold');

Hexadecimale codes vind je via kleurpickers in Photoshop, Figma of op websites zoals coolors.co.

Willekeurige kleuren

Met random(255) genereer je een willekeurige kleurwaarde tussen 0 en 255:

// Willekeurige kleur
fill(random(255), random(255), random(255));

// Willekeurige grijstint
let grijs = random(255);
fill(grijs);

// Willekeurige kleur met vaste transparantie
fill(random(255), random(255), random(255), 180);

Tips en valkuilen

background() met transparantie

Een krachtige creatieve truc: gebruik background() met een lage alpha-waarde in draw(). In plaats van het canvas volledig te wissen, leg je een semi-transparante laag over het vorige frame. Het resultaat: een trailing- of ghosteffect.

function draw() {
  background(0, 15);  // Zwart met lage alpha - spooreffect
  fill(255);
  circle(mouseX, mouseY, 30);
}

Hoe lager het getal, hoe langer het spoor blijft hangen. Probeer waarden tussen 5 en 30.

Lijnen en punten hebben geen fill

Een veelgemaakte beginnersfout: fill() aanroepen voor een line() of point(). Dat heeft geen effect - lijnen en punten reageren alleen op stroke() en strokeWeight().

stroke(255, 0, 0);
strokeWeight(3);
line(10, 10, 200, 200);  // Rode lijn van 3 pixels

// fill() doet hier niets:
fill(0, 255, 0);
point(100, 100);  // Alleen stroke telt

Heel dunne lijnen met sub-pixel waarden

strokeWeight() accepteert ook decimale waarden. Met strokeWeight(0.25) of strokeWeight(0.5) maak je haardunne lijnen - ideaal voor fijne rasters of generatieve patronen.

strokeWeight(0.25);
for (let x = 0; x < width; x += 4) {
  line(x, 0, x, height);  // Haardun raster
}

Kleurmodus

Standaard gebruikt p5.js RGB. Je kunt ook HSB (Hue, Saturation, Brightness) gebruiken:

colorMode(HSB);
fill(0, 100, 100);    // Rood (hue=0)
fill(120, 100, 100);  // Groen (hue=120)
fill(240, 100, 100);  // Blauw (hue=240)

HSB is handig voor kleurwielen en vloeiende kleurovergangen.

Kleur Variabelen

Je kunt kleuren opslaan in variabelen:

let rood = color(255, 0, 0);
let blauw = color(0, 0, 255);

function draw() {
  fill(rood);
  ellipse(100, 200, 50, 50);
  
  fill(blauw);
  ellipse(300, 200, 50, 50);
}
