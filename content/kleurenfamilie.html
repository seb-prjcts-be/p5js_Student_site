Stel je voor: je gaat schilderen en je kiest één verfpot als startpunt. Daarna mag je alleen kleuren gebruiken die er dicht bij liggen. Zo krijg je een samenhangend schilderij - geen wilde chaos. Dat doet deze code ook, maar dan digitaal.

Hoe werkt kleur in p5.js?

Elke kleur bestaat uit drie getallen tussen 0 en 255:

  • R - hoeveel rood
  • G - hoeveel groen
  • B - hoeveel blauw
fill(255, 0, 0)     // puur rood
fill(200, 180, 190) // zacht roze-achtig

Als je alle drie de kanalen volledig willekeurig kiest, krijg je kleurchaos. De truc is om de willekeur te beperken tot een klein venster per kanaal.

De truc in drie stappen

Stap 1 - kies een startpunt per kleurkanaal

let startR = floor(random(0, 155));
let startG = floor(random(0, 155));
let startB = floor(random(0, 155));

Waarom tot 155 en niet 255? Omdat we straks nog 100 optellen. 155 + 100 = 255 - zo blijven we altijd binnen het bereik.

Stap 2 - bepaal het bereik

// Het bereik loopt van startR tot startR + 100
// Voorbeeld: startR = 80 → rood varieert tussen 80 en 180

Stap 3 - kleur elk vakje willekeurig, maar binnen het bereik

fill(
  random(startR, startR + 100),
  random(startG, startG + 100),
  random(startB, startB + 100)
);

Elk vakje krijgt een andere kleur - maar nooit té anders. Ze zitten allemaal in dezelfde buurt.

Waarom is dit slim?

Vergelijk de twee aanpakken:

// Zonder beperking - kleurchaos
fill(random(255), random(255), random(255));

// Met beperking - kleurenfamilie
fill(
  random(startR, startR + 100),
  random(startG, startG + 100),
  random(startB, startB + 100)
);

Met de beperking zitten alle kleuren in dezelfde toonbuurt. Dat is precies wat textielontwerpers doen: ze kiezen een palet van tonen die dicht bij elkaar liggen, zodat elk draadje iets anders is maar het weefsel toch harmonieus oogt. In de mode-industrie heet dat een colorway.

Een raster tekenen met een kleurenfamilie

De dubbele for-lus tekent het raster: rij per rij, kolom per kolom. Elke cel krijgt een eigen kleur, maar altijd uit dezelfde familie:

let startR, startG, startB;
let celBreedte = 60;
let celHoogte = 60;

function setup() {
  createCanvas(600, 400);
  noStroke();
  nieuweKleurenfamilie();
}

function nieuweKleurenfamilie() {
  startR = floor(random(0, 155));
  startG = floor(random(0, 155));
  startB = floor(random(0, 155));
}

function draw() {
  for (let y = 0; y < height; y += celHoogte) {
    for (let x = 0; x < width; x += celBreedte) {
      fill(
        random(startR, startR + 100),
        random(startG, startG + 100),
        random(startB, startB + 100)
      );
      rect(x, y, celBreedte, celHoogte);
    }
  }
  noLoop(); // eén keer tekenen is genoeg
}

function mousePressed() {
  nieuweKleurenfamilie();
  loop(); // opnieuw tekenen
}
