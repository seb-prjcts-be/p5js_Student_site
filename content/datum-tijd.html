p5.js heeft ingebouwde functies om de huidige datum en tijd op te vragen. Daarmee kun je klokken bouwen, countdowns maken of je sketch aan het moment van de dag koppelen.

Kernpunten
  • Tijd opvragen met year(), month(), day(), hour(), minute(), second()
  • Voorloopnullen toevoegen met nf(waarde, 2)
  • Tijd vertalen naar hoeken via map() voor analoge klokken
  • Vloeiende wijzers via millis() in plaats van second()
  • John Maeda's 12 o'clocks als inspiratie om een klok zelf te herdenken

Datum en tijd opvragen

Zes functies geven je de huidige systeemtijd - ze worden elke keer opnieuw uitgevoerd als je ze aanroept:

function draw() {
  let j = year();    // bijv. 2026
  let mo = month();  // 1 – 12
  let d = day();     // 1 – 31

  let u = hour();    // 0 – 23
  let m = minute();  // 0 – 59
  let s = second();  // 0 – 59
}

Combineer ze voor een digitale klok:

function draw() {
  background(20);
  fill(255);
  textSize(48);
  textAlign(CENTER, CENTER);

  // Voeg een nul toe als getal kleiner dan 10
  let u = nf(hour(), 2);
  let m = nf(minute(), 2);
  let s = nf(second(), 2);

  text(u + ":" + m + ":" + s, width / 2, height / 2);
}
Tip

nf(waarde, 2) zorgt voor een vaste breedte van 2 cijfers - zodat 9 wordt weergegeven als 09.

Tijd omzetten naar een hoek

Een analoge klok werkt door seconden, minuten en uren te vertalen naar hoeken. Gebruik map() om van een tijdbereik naar een hoekbereik te gaan.

function setup() {
  createCanvas(400, 400);
  angleMode(DEGREES);
}

function draw() {
  background(20);
  translate(width / 2, height / 2);

  let u = hour() % 12;   // 0 – 11
  let m = minute();       // 0 – 59
  let s = second();       // 0 – 59

  // -90 zodat 0 bovenaan staat (12 uur) in plaats van rechts
  let sHoek = map(s, 0, 60, 0, 360) - 90;
  let mHoek = map(m, 0, 60, 0, 360) - 90;
  let hHoek = map(u + m / 60, 0, 12, 0, 360) - 90;

  // Secundenwijzer
  stroke(255, 60, 60);
  line(0, 0, cos(sHoek) * 130, sin(sHoek) * 130);

  // Minutenwijzer
  stroke(200);
  strokeWeight(3);
  line(0, 0, cos(mHoek) * 100, sin(mHoek) * 100);

  // Urenwijzer
  stroke(255);
  strokeWeight(5);
  line(0, 0, cos(hHoek) * 65, sin(hHoek) * 65);
}

Vloeiende secondenwijzer

second() springt elke seconde een stap. Voor een vloeiende beweging gebruik je millis():

// millis() geeft milliseconden → deel door 1000 voor seconden
let secVloeiend = (millis() / 1000) % 60;
let sHoek = map(secVloeiend, 0, 60, 0, 360) - 90;

John Maeda - 12 o'clocks (1996)

Weetje

In 1996 maakte de Amerikaanse ontwerper en programmeur John Maeda een reeks van twaalf kleine programma's - elk een andere manier om de tijd te tonen. Hij noemde ze 12 o'clocks. Geen van de twaalf lijkt op een gewone klok: stippen verschuiven, cirkels pulseren, lijnen vormen patronen. Maeda's stelling: een klok is een interface voor tijd en een interface kan eindeloos opnieuw worden uitgevonden. 12 o'clocks was een van de vroegste voorbeelden van code als ontwerpmedium - Maeda werkte in Java, jij doet het in p5.js.

De uitdaging: maak jouw eigen klok

Maak een sketch die de huidige tijd toont - maar doe het op een manier die nergens op lijkt wat je al eerder gezien hebt.

Wat mag je gebruiken:

  • Vormen, lijnen, punten, curves
  • Kleur, transparantie, verloop
  • Beweging, rotatie, schaal
  • Rasters, grids, herhaling met for
  • Willekeur via random() of noise()
  • Tekst - al dan niet als cijfers

Wat moet het doen:

  • De huidige tijd correct weergeven (uren, minuten, seconden)
  • Leven in de loop - animatie is niet verplicht, maar het mag
  • Iets tonen wat je aan Maeda zou durven laten zien

Enkele ideeën om los te komen van de standaardklok:

  • Gebruik de tijd als frequentie voor sin() of cos()
  • Laat het aantal seconden het aantal stippen of vormen bepalen
  • Gebruik kleur of transparantie om uren en minuten te coderen
  • Bouw een raster waarbij vakjes aan en uit gaan op basis van de tijd
  • Laat de achtergrondkleur langzaam verschuiven over de dag
