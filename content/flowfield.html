Een flowfield is een onzichtbaar stromingsveld over het canvas. Op elk punt bepaalt noise() een hoek, en lijnen, deeltjes of vormen volgen die hoeken. Het resultaat: samenhangende golven, wervelingen en stromen die levend aanvoelen.

Kernpunten
  • Wat een flowfield is: een raster van hoeken, gegenereerd met noise()
  • map() om noise-waarden (0-1) om te zetten naar hoeken (0-360)
  • translate() en rotate() om elke cel lokaal te draaien
  • push() en pop() om transformaties te isoleren
  • Animatie door een t-variabele door de noise te laten schuiven

Wat is een flowfield?

Een flowfield is een gebied dat een noise()-functie gebruikt om samenhangende tekeningen of animaties te genereren. Stel je voor: een onzichtbare wind waait over het canvas, en op elk punt heeft die wind een andere richting. We gebruiken map() om de waarden tussen 0 en 1 om te zetten tot iets bruikbaars, bijvoorbeeld een rotatie van 0 tot 360.

Basisvoorbeeld

We lopen met een dubbele for-loop over een raster van cellen. Per cel vragen we een noise-waarde, zetten die om naar een hoek, en tekenen een klein kruisje dat volgens die hoek draait.

let cell = 15;
let s = 300;
let t = 0;

function setup() {
  createCanvas(400, 400);
  angleMode(DEGREES);
  rectMode(CENTER);
}

function draw() {
  background(245);
  translate(60, 60);
  for (let y = cell; y < s - 10; y += cell) {
    for (let x = cell; x < s - 20; x += cell) {
      let n = map(noise(x * 0.01 + t, y * 0.01 + t), 0, 1, 0, 360);
      push();
        translate(x + cell / 2, y + cell / 2);
        rotate(n);
        line(0, cell, cell, cell);
        line(cell / 2, cell / 2, cell / 2, cell);
      pop();
    }
  }
  t += 0.008;
}

Experimenteer zelf

Pas de cell-grootte aan, of de stap t += 0.008, en kijk hoe de stroming verandert.

Variaties

  • Vervang de kruisjes door korte lijnen of pijlen die mee stromen.
  • Laat deeltjes op willekeurige startposities elke frame een klein stukje opschuiven volgens de lokale hoek.
  • Gebruik strokeWeight() om de lijndikte te laten variëren met de noise-waarde.
  • Combineer met een kleurenschaal: zet de hoek ook om naar een hue in HSB-modus.

Flowfields werken fantastisch met boids-achtige agents: geef elk deeltje een positie en laat het elke frame versnellen in de lokale hoek van het flowfield.
