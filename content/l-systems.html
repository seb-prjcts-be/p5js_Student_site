Een L-systeem (Lindenmayer-systeem) is een taal van regels die strings herschrijven. Herhaal de regels een paar keer en je krijgt complexe, organische vormen: bomen, planten, fractals.

Hoe werkt het?

Je begint met een axioma — een startstring — en vervangt elke letter herhaaldelijk via regels.

Axioma:  F
Regel:   F → F+F−F−F+F

Iteratie 0:  F
Iteratie 1:  F+F−F−F+F
Iteratie 2:  F+F−F−F+F + F+F−F−F+F − F+F−F−F+F − F+F−F−F+F + F+F−F−F+F
...

Dan teken je de string met turtle graphics:

  • F — ga vooruit en teken een lijn
  • + — draai rechts met de hoek
  • - — draai links met de hoek
  • [ — sla positie op (push())
  • ] — herstel positie (pop())

Plantmodel

Met het [ en ] mechanisme kun je vertakkingen maken:

Axioma: X
Regels: X → F+[[X]-X]-F[-FX]+X
        F → FF
Hoek:   25°

In p5.js gebruik je push() en pop() als schildpad-geheugen:

for (let c of sentence) {
  if (c === "F") { line(0, 0, len, 0); translate(len, 0); }
  else if (c === "+") rotate(radians(hoek));
  else if (c === "-") rotate(radians(-hoek));
  else if (c === "[") push();  // Onthoud positie
  else if (c === "]") pop();   // Herstel positie
}
