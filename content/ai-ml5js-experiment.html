Kies je model

ml5.js biedt verschillende modellen die elk een ander soort input verwerken. Kies het model dat het best past bij jouw idee:

Model Input Wat het doet Geschikt voor
BodyPose Webcam Herkent lichaamspunten (17 keypoints) Bewegingsinteractie, dans, sport
HandPose Webcam Herkent handpunten (21 per hand) Gebaren, tekenen in de lucht, besturing
FaceMesh Webcam Detecteert 468 gezichtspunten Filters, emotiedetectie, maskers
ImageClassifier Webcam / afbeelding Herkent objecten in beeld Objectherkenning, sorteerspellen
SoundClassifier Microfoon Herkent geluidstypen Geluidgestuurde kunst, spraakinteractie

Ideeën ter inspiratie

Hier zijn enkele projectideeën per model - maar het mooiste is als je je eigen richting vindt.

Met BodyPose

  • Een digitale schaduw die je bewegingen volgt maar met vertraging reageert
  • Deeltjes die worden aangetrokken door je handen en afgestoten door je hoofd
  • Een instrument dat je bespeelt door te bewegen - positie bepaalt toonhoogte en volume

Met HandPose

  • Tekenen in de lucht: je vingertop laat een spoor achter op het canvas
  • Een gebarentolk: herken eenvoudige handgebaren en toon het bijbehorende woord
  • Een virtuele marionet die reageert op je vingerbewegingen

Met FaceMesh

  • Een digitaal masker dat meeschuift met je gezicht
  • Abstracte kunst die reageert op gezichtsexpressies - glimlach maakt kleuren warmer
  • Een spiegel die je gezicht vertekent op basis van emotie

Met ImageClassifier

  • Een spel waarbij je objecten voor de camera moet houden die het model herkent
  • Een poëziegenerator: het model herkent een object en genereert er een haiku over

Met SoundClassifier

  • Een visualisatie die anders reageert op spreken, klappen en fluiten
  • Een interactieve installatie waar stilte en geluid twee verschillende werelden creëren

Stappenplan

Een vrij experiment werkt het best met wat structuur. Volg dit stappenplan:

  1. Kies een model - Welke input (camera, microfoon) wil je gebruiken?
  2. Begin simpel - Krijg eerst de basisdetectie werkend. Teken de ruwe data op het scherm.
  3. Voeg visuele laag toe - Vervang de ruwe punten door vormen, kleuren of patronen die je aanspreken.
  4. Maak het interactief - Laat de visuele output reageren op de AI-data op een manier die betekenis heeft.
  5. Verfijn en itereer - Pas kleuren, timing, grootte en compositie aan tot het voelt als iets van jou.

Code-sjabloon: HandPose

Als startpunt voor je experiment kun je dit sjabloon gebruiken. Het detecteert je hand en tekent cirkels op de vingerpunten.

let handPose;
let video;
let handen = [];

async function setup() {
  createCanvas(640, 480);
  video = createCapture(VIDEO);
  video.size(640, 480);
  video.hide();

  // Wacht tot het HandPose-model geladen is
  handPose = await ml5.handPose();
  handPose.detectStart(video, ontvangHanden);
}

function ontvangHanden(resultaten) {
  handen = resultaten;
}

function draw() {
  image(video, 0, 0);

  for (let i = 0; i < handen.length; i++) {
    let hand = handen[i];
    for (let j = 0; j < hand.keypoints.length; j++) {
      let punt = hand.keypoints[j];
      fill(255, 0, 100);
      noStroke();
      circle(punt.x, punt.y, 12);
    }
  }
}

Pas dit aan: verander de vormen, voeg trails toe, experimenteer met kleuren die reageren op de positie van je vingers.

Tips voor het experiment

  • Test vroeg - Laat je code vaak draaien. Kleine stappen werken beter dan grote sprongen.
  • Gebruik AI als hulp - Plak je code in een chatbot en vraag: "Hoe voeg ik een trail-effect toe aan deze punten?"
  • Documenteer je proces - Maak screenshots of korte video's van tussenresultaten. Het proces is minstens zo waardevol als het eindresultaat.
  • Combineer modellen - Gevorderden kunnen twee modellen tegelijk gebruiken, bijvoorbeeld BodyPose én SoundClassifier.
  • Denk na over presentatie - Hoe toon je dit aan anderen? Werkt het als installatie? Als schermwerk? Als performance?
