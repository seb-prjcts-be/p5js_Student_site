p5.js kan meer dan alleen 2D! Met de WEBGL-renderer maak je 3D graphics: vormen in de ruimte, rotaties, belichting en materialen.

WEBGL Canvas

Om 3D te gebruiken, geef je WEBGL mee als derde argument:

function setup() {
  createCanvas(600, 400, WEBGL);
}

function draw() {
  background(245);
  box(100);  // Standaard kubus in het midden
}

In WEBGL-mode staat het middelpunt van het canvas op coördinaat (0, 0, 0). Gebruik translate() om objecten te verplaatsen.

3D Vormen

p5.js heeft ingebouwde 3D vormen:

  • box(breedte, hoogte, diepte) — kubus of doos
  • sphere(straal, detail) — bol
  • cylinder(straal, hoogte) — cilinder
  • cone(straal, hoogte) — kegel
  • torus(straal, buisdikte) — donut-vorm
  • plane(breedte, hoogte) — plat vlak
function draw() {
  background(30);
  rotateY(frameCount * 0.01);

  // Drie vormen naast elkaar
  push();
    translate(-150, 0, 0);
    box(70);
  pop();

  push();
    translate(0, 0, 0);
    sphere(45);
  pop();

  push();
    translate(150, 0, 0);
    torus(40, 15);
  pop();
}

Transformaties in 3D

Naast translate() en scale() heb je drie rotatierichtingen:

function draw() {
  background(245);

  rotateX(frameCount * 0.01);  // Kantelt voor/achter
  rotateY(frameCount * 0.02);  // Draait links/rechts
  rotateZ(frameCount * 0.005); // Rolt

  box(100);
}

Gebruik altijd push() / pop() om transformaties te isoleren:

function draw() {
  background(245);

  push();
    translate(-120, 0, 0);
    rotateY(frameCount * 0.02);
    box(80);
  pop();

  push();
    translate(120, 0, 0);
    rotateX(frameCount * 0.03);
    sphere(50);
  pop();
}

Belichting

Zonder licht zien 3D vormen er plat uit. p5.js heeft verschillende lichtbronnen:

ambientLight() — omgevingslicht

function draw() {
  background(0);
  ambientLight(100);  // Waarde 0-255: hoe helder het omgevingslicht
  sphere(100);
}

directionalLight() — richtingslicht

function draw() {
  background(0);
  // directionalLight(kleur, richting als vector)
  directionalLight(255, 255, 255, -1, 1, -1);
  sphere(100);
}

pointLight() — puntlicht

function draw() {
  background(0);
  // Licht dat vanuit één punt schijnt
  pointLight(255, 200, 100, mouseX - width/2, mouseY - height/2, 200);
  sphere(100);
}

Combinatie

function draw() {
  background(0);
  ambientLight(40);                             // Zwak omgevingslicht
  directionalLight(255, 240, 200, 1, 1, -1);   // Zonlicht-effect
  pointLight(100, 150, 255, -200, -100, 100);   // Blauw accent
  sphere(100);
}

Materialen

Materialen bepalen hoe een vorm reageert op licht:

normalMaterial() — kleurt op basis van normaalvector

function draw() {
  background(0);
  normalMaterial();  // Geen extra parameters nodig
  box(100);
}

ambientMaterial() — reageert op omgevingslicht

function draw() {
  background(0);
  ambientLight(200);
  ambientMaterial(255, 100, 0);  // Oranje materiaal
  sphere(100);
}

specularMaterial() — glanzend oppervlak

function draw() {
  background(0);
  ambientLight(60);
  directionalLight(255, 255, 255, 0, 0, -1);
  specularMaterial(255);  // Glanzend wit
  shininess(50);          // Hoe scherp de glinstering is
  sphere(100);
}

Camera Instellen

Met camera() bepaal je vanwaar je kijkt:

function draw() {
  background(0);

  // camera(oogX, oogY, oogZ, doelX, doelY, doelZ, upX, upY, upZ)
  camera(0, -200, 400,   // Positie van de camera
         0, 0, 0,        // Naar welk punt je kijkt
         0, 1, 0);       // "Up" richting

  box(100);
}

// Eenvoudiger: orbitControl() voor muis-gestuurd rondkijken
function draw() {
  background(30);
  orbitControl();  // Klik + sleep om te roteren, scroll om in/uit te zoomen
  box(100);
}

Shaders laden

Shaders zijn programma's die rechtstreeks op de GPU draaien. Laad ze met await loadShader() in async function setup():

let mijnShader;

async function setup() {
  createCanvas(600, 400, WEBGL);
  mijnShader = await loadShader('shader.vert', 'shader.frag');
}

function draw() {
  shader(mijnShader);
  mijnShader.setUniform('uTime', millis() / 1000);
  rect(-width / 2, -height / 2, width, height);
}

Dit is een geavanceerde functie — handig als je WEBGL volledig wilt aansturen. Shaders zijn kleine programmaatjes (GLSL) die bepalen hoe elk pixel eruitziet.
